Encore is an open-source development platform and backend framework designed for building type-safe, cloud-native distributed systems with declarative infrastructure. Written primarily in Go, it simplifies microservice creation and API development, offering built-in support for local development with a CLI that manages infrastructure, including tracing and a service catalog, and integrates with cloud platforms like AWS and GCP through optional features. Key functionalities include defining services, APIs, databases, cron jobs, and Pub/Sub as type-safe objects, automating infrastructure provisioning and deployment, and generating API documentation and client libraries. This framework targets backend developers seeking to streamline development workflows and improve the scalability of their applications.

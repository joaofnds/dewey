`yt-dlp/yt-dlp` is a feature-rich command-line application written in Python designed for downloading audio and video content from a wide range of websites. It functions as a video downloader, supporting thousands of sites and offering advanced capabilities, including format selection, subtitle handling, metadata modification, and integration with SponsorBlock. The project provides extensive configuration options, plugin support, and is built upon a dependency stack that includes Python 3.9+, ffmpeg, and various networking and metadata libraries. This standalone CLI application serves as a replacement for youtube-dl and is aimed at users who want to download media from online sources.

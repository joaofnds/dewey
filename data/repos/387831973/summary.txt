`refactoring.nvim` is a Lua plugin for Neovim that provides a library of refactoring and debugging tools for various programming languages. It offers functionalities mirroring common refactoring operations from Martin Fowler's "Refactoring" book, including extract function, inline function, extract variable, and inline variable, along with debug features such as printf insertion and print variable statements. Configured via Lua, the plugin supports languages including TypeScript, JavaScript, Lua, C/C++, Golang, Python, Java, PHP, Ruby, and C#, and integrates with Neovim's features via Ex commands, a Lua API, built-in selection, and a Telescope extension. It is a developer productivity tool, designed to enhance code maintainability and debugging within the Neovim editor.

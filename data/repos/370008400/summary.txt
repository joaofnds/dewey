Scittle provides a means to execute Clojure(Script) code directly within web browser script tags, leveraging the Small Clojure Interpreter (SCI). This project functions as a developer productivity tool, enabling in-browser ClojureScript execution. Key functionalities include allowing users to run Clojure code within HTML via SCI, with available tasks for development builds, production artifact creation, and Github pages updates. It is a Clojure library, specifically targeting developers working with Clojure and ClojureScript for front-end and possibly full-stack web development.

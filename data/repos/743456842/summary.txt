Verso is a web browser application built using the Rust programming language and leverages the Servo web engine. The project aims to explore and implement multi-view and multi-window capabilities within the Servo engine to evolve into a mature browser. Key functionalities include web page rendering and navigation, with development currently focused on core browser features. The project emphasizes Flatpak and Nix for build management and packaging, targeting Linux, macOS, and Windows platforms, along with nightly releases.

http-kit is a high-performance, event-driven HTTP client and server library for Clojure, designed for building concurrent and asynchronous web applications. It provides a Ring-compatible API, enabling seamless integration with existing Clojure middleware and libraries. Key functionalities include efficient HTTP handling, support for WebSockets and HTTP long-polling/streaming, and a small footprint with zero dependencies. This library is primarily intended for Clojure developers seeking a lean and performant solution for both client and server-side HTTP communication.

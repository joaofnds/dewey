`jsonapi-resources` is a Ruby library designed to facilitate the development of JSON:API compliant servers within the Rails framework. It provides a resource-focused architecture, enabling developers to define resources, their attributes, and relationships, thereby generating compliant APIs with custom routes, controllers, and serializers. The library integrates with Rails 5.1+ and supports both ActiveRecord models and custom Ruby objects. It is a backend library targeted at Ruby on Rails developers building JSON:API-based web services.

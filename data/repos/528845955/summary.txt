`notifier.nvim` is a Lua plugin designed to provide a non-intrusive notification system for the Neovim text editor. It enhances the user experience by displaying notifications from `vim.notify` and LSP servers within a floating window, with configurable options for title, icon, and content. The plugin features commands for clearing and replaying notifications, and it leverages highlight groups for customizable visual styling. Primarily a developer productivity tool, it aims to improve the way Neovim users receive and manage informational updates.

Goose is a Clojure library designed for reliable background job processing, supporting asynchronous and batch processing workflows. It provides native integration with RabbitMQ and Redis for message queuing, alongside features such as scheduled and cron jobs, error handling with retries, and a jobs management console. This library is architected to be pluggable, extensible via middlewares, and scalable, making it suitable for production environments. It targets Clojure developers building applications requiring asynchronous task execution.

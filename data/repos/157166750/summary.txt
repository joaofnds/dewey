Lens is a standalone desktop application, a Kubernetes Integrated Development Environment (IDE), designed for cloud-native software development and operations. It provides a unified, context-aware user interface for managing and troubleshooting Kubernetes workloads. The application allows users to learn and use Kubernetes, boost team productivity, and reduce the tool overhead for cloud-native development. Targeted at Developers and DevOps Engineers. The retired source code is still available in the 'master'-branch.

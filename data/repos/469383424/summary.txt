`leap.nvim` is a Neovim plugin that significantly enhances text navigation within the editor, providing a fast and uniform method for jumping to any visible screen position. It implements a two-character search pattern with a preview of target labels, improving upon the functionality of `vim-sneak`. Written in Fennel, the plugin's key features include the ability to reach any target with minimal keystrokes, support for repeated motions, and the option for remote operations via the `remote` API, including the creation of custom text objects. `leap.nvim` is a plugin designed for Vim users, aiming to improve their efficiency and ease of use in modal text editing.

This repository is a curated, open-source list (an "awesome list") of mechanical keyboard resources. It provides a categorized compilation of projects, tools, and tutorials related to mechanical keyboards, including staggered, ortholinear, split, and firmware-related topics. Built using Astro, a static site generator, and deployed on Netlify, it serves as a central hub for mechanical keyboard enthusiasts and developers. Contributions are welcomed through Markdown file editing and pull requests.

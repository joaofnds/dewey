Neanderthal is a high-performance Clojure matrix computation library designed for numerical linear algebra and scientific computing applications. Built on optimized BLAS and LAPACK native libraries, it provides fast matrix operations, factorizations, and linear algebra computations for both CPU and GPU environments through CUDA and OpenCL support. The library targets data scientists, machine learning engineers, and scientific computing developers who need efficient matrix calculations within the Clojure ecosystem. It addresses performance bottlenecks in numerical computations by leveraging hardware acceleration and vectorization, making it suitable for deep learning, numerical analysis, and high-performance computing workloads that require intensive matrix operations while maintaining Clojure's functional programming paradigms.

Neanderthal is a high-performance Clojure library designed for fast matrix and linear algebra computations. Leveraging optimized native libraries like BLAS and LAPACK, it provides functionalities for matrix calculations, including operations and factorization, targeting both CPU and GPU architectures. This library offers a Clojure API for numerical computing, catering to users focused on scientific computing and high-performance computing tasks. It can be considered a Clojure library with support for CUDA and OpenCL.

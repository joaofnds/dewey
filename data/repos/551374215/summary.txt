CozoDB is a transactional, relational database leveraging Datalog for querying, designed as an embeddable solution with a focus on graph data and algorithms.  Written in Rust, it provides features including time travel, support for various storage backends like RocksDB, SQLite, and in-memory, and offers graph database capabilities enhanced with vector search using HNSW indices integrated within Datalog. The project further offers multi-platform support, client-server mode, and multiple language bindings through wrappers for various environments, including Python, NodeJS, and the browser.  This project is a standalone database system, and offers the possibility to be used as a library.

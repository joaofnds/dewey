org-wild-notifier.el is an Emacs Lisp package that provides desktop notification alerts for org-agenda TODO entries and scheduled events within the Emacs org-mode ecosystem. Built on the alert notification framework, it targets Emacs power users and productivity enthusiasts who rely on org-mode for task management and scheduling. The package offers configurable alert timing, keyword/tag-based filtering through whitelists and blacklists, and predicate-based custom filtering functions. It operates in both automatic timer-based and manual notification modes, solving the problem of missed deadlines and appointments by delivering timely desktop notifications. Key features include customizable alert intervals via org properties, integration with existing org-agenda workflows, and extensible notification styling through the underlying alert package infrastructure.

`cuelsp` is a Language Server Protocol (LSP) implementation for the CUE configuration language, integrating with the Dagger build tool. It provides features such as definition jumping, doc hover support, and loading of CUE files, with development utilizing Dagger for linting, testing, and binary builds. The project is written in Go and targets developers using CUE and Dagger within code editors like VS Code and Vim through standalone binary or Dagger CLI integration. It is a standalone application providing language server functionality.

`diagrams` is a Python library enabling "Diagram as Code" for cloud system architecture prototyping and visualization. It allows users to define cloud infrastructure diagrams using Python, supporting major providers like AWS, Azure, GCP, and Kubernetes, along with on-premises and SaaS components.  The library leverages Graphviz for diagram rendering and facilitates version control for architectural diagrams. This developer-focused tool is primarily a Python library, targeting developers and system architects designing and documenting cloud infrastructure.

`cq` is a Clojure-based command-line data processor designed for transforming various data formats including JSON, YAML, EDN, XML, and CSV. It leverages Clojure's functional programming capabilities, offering a query language based on Clojure syntax with features like threading macro redirection and reader macros for concise query construction. The application integrates several Clojure libraries, such as Specter, Medley, and Camel-Snake-Kebab, providing powerful data manipulation and transformation capabilities. It is a standalone CLI application that targets developers and data engineers needing a flexible tool for processing and querying diverse data sources directly from the command line.

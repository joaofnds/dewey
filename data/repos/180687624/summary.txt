Trivy is a security scanner designed to identify vulnerabilities, misconfigurations, secrets, and software bill of materials (SBOM) in container images, filesystems, Git repositories, Kubernetes clusters, and virtual machine images. Written in Go, it scans for OS package vulnerabilities, IaC issues, sensitive information, and license compliance. This standalone command-line application (CLI) integrates with popular platforms and supports a wide range of programming languages and operating systems, making it a versatile tool for DevSecOps and infrastructure security professionals.

`jason` is a high-performance JSON parser and generator library written in pure Elixir. It provides functionalities for encoding Elixir data structures to JSON and decoding JSON strings into Elixir terms, adhering to RFC 8259 and ECMA 404 standards. The library is designed for speed, aiming for performance comparable to C-based JSON parsers, and integrates with other Elixir libraries such as Postgrex, Ecto, and Plug. This library offers features like JSON fragment injection and custom encoder implementations and serves as a developer productivity tool, enabling efficient JSON handling within Elixir applications.

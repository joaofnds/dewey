This repository provides a collection of monospaced bitmap fonts in BDF format, derived from IBM text mode and system fonts, encompassing extended Latin, Greek, Cyrillic, and Hebrew scripts. The project enables users to install and utilize these fonts within applications supporting Xft or XLFD, with configuration options for installation directories via a `configure` script and instructions for font cache updates. Offering a variety of font sizes and styles, the project functions as a font library, targeting users who require access to classic retro-styled fonts for various applications, including terminal emulators and retro computing projects. Installation is facilitated with an Arch Linux AUR package.

Jitsi Meet is a TypeScript-based, open-source, standalone web application and SDK for secure, scalable video conferencing. It provides features including HD audio/video, content sharing, mobile applications, and integrations via web and native SDKs. The project supports all modern browsers and facilitates features like raise hands, reactions, polls, and virtual backgrounds. Deployments are simplified through Debian packages and Docker setups, targeting developers and organizations seeking to implement video conferencing solutions, and also provides options for white-labeling and embedding within web applications.

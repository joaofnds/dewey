WBOT is a standalone, command-line tool (CLI) application written in JavaScript using Node.js, designed to automate interactions with WhatsApp Web. It functions as a bot, enabling users to respond to messages, download media, and implement custom responses based on configurable keywords. Leveraging the Puppeteer library for browser automation, the application offers features like customizable JSON configurations, multi-instance support, and the ability to inject custom JavaScript. Its primary target audience appears to be individuals seeking a simple, free tool to automate WhatsApp interactions for personal use.

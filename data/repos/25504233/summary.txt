`alchemist.el` is an Emacs plugin providing comprehensive integration for Elixir development. It offers a suite of features including Elixir code compilation and execution, inline evaluation and macro expansion, documentation and definition lookup, smart code completion, and project management capabilities, with specific support for Phoenix projects. Built using Emacs Lisp, the plugin integrates with the Emacs ecosystem, particularly with `company-mode`. It is a developer productivity tool aimed at Elixir developers working within the Emacs environment.

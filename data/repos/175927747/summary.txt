Storeon is a tiny (185 bytes) JavaScript library providing a Redux-like, event-based state management solution primarily targeted at frontend development within React, Preact, Angular, Vue, and Svelte applications. It enables efficient state management with a focus on speed and modularity, offering features such as state change tracking for optimized re-rendering, Redux-like reducers implemented as event listeners, and modular API design to separate business logic. The library is designed for use within web applications, and it supports TypeScript, dev tools integration, and provides utilities like `@storeon/router` and `@storeon/localstorage` to extend its functionality, making it a library for developers building interactive web frontends.

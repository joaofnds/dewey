`Corgi` is a command-line workflow manager, written in Go, designed to organize and automate repetitive command-line operations by encapsulating them into reusable snippets. It enables users to create, list, execute, edit, and share command snippets, including support for multi-line commands and template fields for dynamic input. Features include interactive snippet selection via fuzzy-finding tools (fzf, peco), and exporting snippets as JSON or shell scripts. This standalone CLI application streamlines repetitive command execution, targeting developers and system administrators seeking to improve command-line productivity.

flyradar is a Rust-based terminal UI for managing and monitoring Fly.io resources, inspired by k9s. It offers an interactive, keyboard-driven interface to interact with your Fly.io apps, VMs, volumes, and secrets from the terminal. The tool provides real-time resource monitoring, quick access to logs with filtering and dumping capabilities, and a focus on operational workflows for existing resources. flyradar relies on the Fly CLI for authentication and its built-in agent for operational functionality. This project is an OSS third-party tool that follows Conventional Commits for commit messages and adheres to best practices in code formatting, linting, and testing. It is a terminal UI application designed for users who manage Fly.io resources.

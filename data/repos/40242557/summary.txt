Lem is a Common Lisp editor/IDE designed for enhanced developer productivity. It provides features including project exploration, project-aware commands, interactive grep and git interfaces, and multiple cursors, with support for Emacs and Vim modes. Leveraging an integrated LSP client, it supports various programming languages, offering a streamlined development experience, especially for Common Lisp projects. This standalone application is available in both ncurses and SDL2 versions for Linux, MacOS, and Windows, catering to developers working on Common Lisp projects.

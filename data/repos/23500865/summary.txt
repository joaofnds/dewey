`hyper` is a low-level, asynchronous HTTP library written in Rust, designed to be a building block for HTTP clients and servers. It provides support for both HTTP/1 and HTTP/2 protocols, emphasizing performance and correctness. Key features include client and server APIs suitable for production use. The library is intended for use in creating higher-level applications and libraries, with examples of downstream usage in frameworks such as `axum` and `warp`.

IOPaint is a Python-based, open-source image inpainting and outpainting tool leveraging state-of-the-art AI models, primarily targeting image manipulation tasks. Its core functionality includes object removal, defect correction, and content replacement, utilizing models such as LaMa and Stable Diffusion. The project features a web user interface, command-line interface for batch processing, and plugin support for functionalities like segmentation and super-resolution. It relies on Python with PyTorch for AI model execution, supports CPU, GPU, and Apple Silicon, and is designed as a standalone application.

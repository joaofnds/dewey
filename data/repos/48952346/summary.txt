The `buttercup/buttercup-browser-extension` repository provides a browser extension designed as a password manager interface. Built with TypeScript, it allows users to interact with password archives managed by the Buttercup desktop application. Core functionalities include auto-filling login forms, saving new credentials, and integrating with various cloud storage services like Dropbox and Google Drive. The extension is designed as a plugin for Chrome, Firefox, Edge, and Brave browsers, enabling secure access to and management of user credentials from within the browser environment. The project is an archived extension with a history of support for password vaulting and password management automation.

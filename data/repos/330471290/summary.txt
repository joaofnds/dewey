PKMDS-Blazor is a web application providing a save editor for Pokémon games, allowing users to modify their save files. Built with Blazor WebAssembly, it leverages .NET (likely .NET Core) and associated ASP.NET technologies for a client-side web application experience. The application is powered by the PKHeX.Core library and supports a wide range of Pokémon game save file formats. It targets Pokémon game enthusiasts seeking to edit their game data within a web browser, functioning offline after initial caching.

`gjson.rs` is a Rust crate designed for fast JSON value retrieval. It provides a simple API with dot notation path syntax for accessing values, including support for wildcards, array indexing, and querying within arrays. Key features include one-line retrieval, JSON Lines parsing, and value type conversions, offering a streamlined approach to working with JSON data. As a library, `gjson.rs` enables developers to efficiently parse and extract data from JSON documents within their Rust applications, potentially for data processing and API interactions.

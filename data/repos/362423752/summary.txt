`folke/which-key.nvim` is a Neovim plugin designed to enhance user discoverability and memorization of keybindings. It displays available keymaps in a popup as the user types, supporting various Neovim modes and offering customizable layouts and sorting options. Built using Lua, the plugin integrates with external icon libraries and includes built-in plugins for marks, registers, and spelling suggestions, thereby improving developer productivity. It is a Neovim plugin intended for users of the Neovim text editor who want to improve their keybinding knowledge and overall editing workflow.

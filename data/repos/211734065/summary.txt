Crush is a command-line shell and a modern programming language, implemented in Rust, designed for both batch and interactive usage. It provides features typically found in modern programming languages, including a type system, closures, and lexical scoping, while maintaining a shell-like syntax.  This standalone application offers functionalities such as shell command execution and programming language constructs for scripting and automation. Build and installation require Rust, and external dependencies such as OpenSSL and Protobuf are necessary for building on macOS.

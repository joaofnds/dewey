The `hasura/graphql-engine` repository provides a high-performance, open-source GraphQL engine designed to deliver instant, real-time GraphQL APIs on top of various data sources. Functionality includes fine-grained access control and the ability to trigger webhooks based on database events. Built primarily in TypeScript, the project supports PostgreSQL, MongoDB, ClickHouse, and MS SQL Server, along with custom business logic integration using connector SDKs. This is a standalone API service project, offering a complete GraphQL API solution and targeting developers building modern applications.

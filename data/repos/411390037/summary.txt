`frowny.el` is an Emacs Lisp package designed to insert "frownies" (emoticons depicting a frowning face) within Emacs text editor buffers. This plugin extends Emacs functionality by providing a simple feature: adding textual representations of a sad face in the user's text. It operates as an extension, enhancing the user experience within the Emacs text editor environment. This project targets Emacs users who want to quickly insert these specific emoticons.

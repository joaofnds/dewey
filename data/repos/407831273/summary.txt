`rueidis` is a high-performance Go library designed as a Redis client, supporting features like auto-pipelining and server-assisted client-side caching to improve Redis interaction efficiency. This library facilitates a developer-friendly command builder for constructing and executing Redis commands, including support for RedisJSON, RedisBloom, and RediSearch. Key functionalities include generic object mapping, distributed locks with client-side caching, Pub/Sub, and integration with OpenTelemetry, targeting Go developers building applications that leverage Redis for caching, data storage, and other related services. It provides various utilities like a client-side caching helpers, stream read capabilities, and options for connecting to Redis clusters, sentinels, and standalone instances.

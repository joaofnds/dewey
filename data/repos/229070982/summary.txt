`k8s-api` is a Clojure library providing a data-driven Kubernetes client, enabling interaction with the Kubernetes API. It facilitates client instantiation with various authentication methods, including token-based and certificate-based approaches. Key functionalities include operation discovery via `explore`, providing detailed information on operations with `info`, and invocation of API calls using `invoke`. This library targets developers working with Kubernetes, providing a Clojure-based interface for managing and interacting with Kubernetes resources.

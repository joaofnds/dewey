clj-github is a Clojure library providing programmatic access to the GitHub REST API, primarily targeting DevOps and software development automation workflows. Built on http-kit for HTTP client functionality, it offers multiple authentication mechanisms including GitHub App installation tokens, personal access tokens, and configurable token functions with credential chaining. The library features automated token management with caching and refresh capabilities, specialized repository management functions, and a functional changeset API for programmatic Git operations like branching, file modifications, and commits. It includes comprehensive testing utilities with mocking capabilities via httpkit.fake integration. Designed for Clojure developers building CI/CD pipelines, repository automation tools, and GitHub integration services requiring reliable API interaction with proper authentication handling.

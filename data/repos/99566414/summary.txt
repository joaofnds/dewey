`reitit` is a high-performance, data-driven routing library for Clojure and ClojureScript, designed primarily for web application development. It provides a flexible, modular architecture that facilitates bi-directional routing, route conflict resolution, and first-class route data, supporting ring, http, pedestal, and frontend integration. Key features include pluggable coercion using malli, schema, and clojure.spec, along with built-in support for OpenAPI and Swagger. This library is a reusable component (a library) for Clojure developers building web applications, offering essential functionality for handling HTTP request routing and related middleware tasks.

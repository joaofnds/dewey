Gitea is a self-hosted, all-in-one software development service providing Git hosting, code review, team collaboration, package registry, and CI/CD capabilities. Written in Go, it supports multiple platforms including Linux, macOS, and Windows. Key features include a web interface for managing repositories, issues, pull requests, and actions, alongside support for Docker registry, Maven and npm registries. Gitea is a standalone web application designed to provide a painless, self-hosted Git service, targeting developers and DevOps teams seeking an alternative to cloud-based platforms like GitHub or GitLab.

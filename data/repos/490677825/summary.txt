Orama is a TypeScript library providing a complete, in-browser, server, or edge network search engine and RAG pipeline, enabling full-text, vector, and hybrid search capabilities in under 2KB. It offers features like typo-tolerance, BM25 ranking, stemming in multiple languages, vector search, and hybrid search combining full-text and vector approaches. Designed for use in web applications and other JavaScript environments, Orama supports data indexing with schema definition, including vector embeddings, and facilitates integration with GenAI chat sessions through a plugin architecture. This library is primarily aimed at developers needing a performant, embeddable search solution.

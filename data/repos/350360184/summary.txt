Cal.com is an open-source scheduling infrastructure web application, functioning as an alternative to Calendly. Built with TypeScript, it leverages a T3 stack, incorporating Next.js, tRPC, Prisma, and Tailwind CSS for a full-stack web experience. Key features include self-hosting capabilities, white-labeling, and an API-driven architecture, providing users with control over data and workflows. The project is a standalone web application designed for anyone needing scheduling solutions.

Argo Rollouts is a Kubernetes controller and set of CRDs designed for progressive delivery, enabling advanced deployment strategies within the Kubernetes ecosystem. It provides blue-green and canary deployments, along with features for experimentation and automated rollbacks, enhancing control and safety during application updates. Built with Go, it integrates with ingress controllers, service meshes, and metric providers to manage traffic shaping and automated analysis of key performance indicators (KPIs). This project functions as a Kubernetes controller and provides integration plugins, targeting system administrators and DevOps engineers focused on streamlining and improving Kubernetes deployment processes.

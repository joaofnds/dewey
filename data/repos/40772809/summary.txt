AdonisJS/core is a TypeScript-first web framework designed for building fullstack web applications and API servers on Node.js. It provides a robust MVC framework with features including testing support, modern tooling, and an ecosystem of official packages, streamlining web development. The framework offers a clean API for building both web applications and microservices, focusing on developer ergonomics and speed. This project is a framework intended for backend developers building web applications using TypeScript and Node.js.

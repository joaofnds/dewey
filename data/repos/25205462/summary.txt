Woo is a high-performance, non-blocking HTTP server library written in Common Lisp. It leverages libev for event handling and aims to be the fastest web server in any language. Key functionalities include handling HTTP requests and responses, supporting SSL via OpenSSL or LibreSSL, and integration with the Clack web application abstraction. This library can be used either directly with `woo:run` or through Clack for web application deployments, and supports worker processes for enhanced performance. It is a backend library intended for Common Lisp developers building web applications.

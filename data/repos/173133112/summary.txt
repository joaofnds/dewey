`GoogleChromeLabs/proxx` is a web-based game of proximity developed with Preact for UI components, PostCSS for styling, and Comlink for web worker handling, and bundled with Rollup. The application features both WebGL and Canvas 2D rendering modes, leveraging IndexedDB for sprite storage, and incorporates a separate gamelogic component running inside a web worker to optimize animation performance. It provides configurable debugging flags via URL parameters, and serves as a standalone, interactive web application accessible on various devices, targeting users with web browsers.

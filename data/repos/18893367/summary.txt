`bleve` is a Go library providing a modern text, numeric, geo-spatial, and vector indexing and search solution. It enables indexing of various Go data structures and JSON with support for multiple field types, including text, numbers, geospatial points, and vectors. Key features include flexible query types, scoring models (tf-idf, bm25), result highlighting, and aggregations/faceting, alongside a command-line interface for index interaction. The library targets developers seeking to incorporate efficient search capabilities within their Go applications, particularly in domains requiring text analysis, geospatial data handling, and vector search functionality.

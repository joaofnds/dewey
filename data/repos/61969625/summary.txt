`just` is a dependency-free JavaScript library providing a collection of small, single-purpose utility modules. These modules address common programming tasks, including array manipulation, object handling, string formatting, number operations, and function composition, all designed for use in PWA or performance-critical scenarios. The library supports both ES and CJS module formats, offers TypeScript definitions, and is targeted towards frontend developers seeking lightweight, modular solutions. It is a library comprised of individual packages, each addressing a specific utility function.

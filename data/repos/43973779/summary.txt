shadow-cljs is a ClojureScript build tool and compiler that streamlines web development workflows for Clojure developers. It provides simplified configuration defaults, seamless npm integration, and supports multiple deployment targets including browser applications, Node.js scripts, React Native, and Chrome extensions. The tool features fast incremental compilation with reliable caching, live reload capabilities for both ClojureScript and CSS, integrated REPL development environment, and code splitting via modules. Built on the JVM and requiring Node.js, it addresses the complexity of ClojureScript compilation and JavaScript ecosystem integration, enabling developers to build modern web applications while leveraging Clojure's functional programming paradigms and maintaining productive development cycles through hot-reloading and interactive programming features.

The `znck/grammarly` repository provides a Visual Studio Code extension implementing a language server for grammar and spell checking, leveraging Grammarly's SDK. Built with TypeScript, this extension integrates grammar checking functionality directly into the VS Code editor. It supports multiple languages, requiring the integration of tree-sitter grammars and language transformers for each supported language. It is a VS Code extension, targeting developers who use VS Code and seek grammar and spelling assistance while writing code or text.

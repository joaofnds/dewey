Gitako is a browser extension developed in TypeScript providing a file tree view for GitHub, Gitee, and Gitea repositories, enhancing developer productivity. Key features include instant file search, support for private repositories and git submodules, intuitive keyboard navigation, and code folding capabilities. The extension integrates directly into GitHub, Firefox, Edge, and Chrome, providing a file navigation experience within their respective web interfaces. As a browser extension, it targets developers interacting with code repositories within their web browsers.

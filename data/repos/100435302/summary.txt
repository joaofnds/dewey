`eglot` is an Emacs Lisp package providing a client for Language Server Protocol (LSP) servers, enabling advanced code editing features within the Emacs environment. It facilitates integration with various language servers, supporting functionalities such as completion, snippets, diagnostics, code actions, symbol hovers, and refactoring operations like renaming and finding definitions. Eglot is an Emacs extension that leverages built-in Emacs functionalities, interacts with external language servers for different programming languages (e.g., C++, Python, JavaScript), and is now incorporated into Emacs core, streamlining feature development and distribution. Its target audience is Emacs users who want to utilize LSP servers for improved coding assistance.

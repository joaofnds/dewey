`rs/zerolog` is a high-performance JSON logging library for Go, designed to minimize allocations and maximize speed. It provides structured logging capabilities, including leveled logging (panic, fatal, error, warn, info, debug, trace), contextual fields, error logging with optional stack traces, hooks, and integration with `context.Context` and `net/http`. This library offers features like sub-loggers, pretty console logging, and CBOR binary encoding, targeting Go developers who need efficient and structured logging in their applications; it is a library.

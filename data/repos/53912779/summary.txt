`Neoformat` is a Neovim plugin designed to format code within the editor using various external formatters. It integrates with formatters for numerous file types, applying them to the current buffer and updating it with the formatted output. The plugin supports custom formatter configuration, including options for arguments, stdin/stdout handling, and environment variables, as well as the ability to run formatters on file save. This plugin is a developer productivity tool, intended for Vim/Neovim users who want an integrated code formatting solution within their editor, providing a consistent code style across different projects and languages via a plugin.

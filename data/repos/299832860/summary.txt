`livekit/livekit` is an open-source, Go-based project providing a scalable WebRTC SFU (Selective Forwarding Unit) media server and comprehensive SDKs for real-time video, audio, and data capabilities. It offers features including simulcast, speaker detection, end-to-end encryption, and advanced networking, along with client SDKs for JavaScript, Swift, Kotlin, Flutter, Unity (WebGL), React Native, and Rust. This project is a standalone application (media server) with accompanying SDKs and tools, targeted towards developers building real-time communication applications, offering both self-hosting and cloud-based deployment options.

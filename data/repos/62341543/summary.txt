Gleam is a statically-typed programming language designed for building scalable and type-safe systems. Implemented in Rust, it compiles to Erlang virtual machine bytecode, enabling its use within the Erlang ecosystem. The project functions as a standalone programming language, including a compiler. Gleam's key features include type safety, indicating its focus on reliable, maintainable code. This project caters to developers aiming to create robust and scalable applications.

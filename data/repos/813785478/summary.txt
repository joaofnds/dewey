SQL Studio is a standalone, Rust-based, command-line application and GUI providing a unified SQL database explorer. It facilitates interaction with various database systems, including SQLite, libSQL, PostgreSQL, MySQL/MariaDB, ClickHouse (partial support), and Microsoft SQL Server. Key features include an overview page with metadata, table metadata views, a query interface with SQL IntelliSense, and infinite scroll row viewing. The project is packaged with prebuilt binaries installable via shell and Powershell scripts, and is also available via Nix and Docker.

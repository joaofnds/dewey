This repository provides a Docker containerized deployment of Pi-hole, a network-wide ad-blocking and DNS server solution. It leverages Docker for simplified installation and management, allowing users to deploy Pi-hole on various platforms. Key functionalities include blocking ads across a network, serving as a DNS server, and offering a web interface for configuration and monitoring. The project utilizes Docker and shell scripting for containerization and management, with a configuration primarily managed via environment variables and volume mounts. It is a standalone application aimed at system administrators and home network users seeking to improve network privacy and control over internet traffic.

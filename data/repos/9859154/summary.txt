This repository hosts the source code and documentation for the `application/vnd.api+json` media type, a specification for building JSON APIs. Primarily serving as a documentation project, it defines the standards for structuring API responses and requests using JSON format. The project is a specification, including the accompanying website hosted on GitHub Pages, and is actively maintained. The development process involves using Ruby, implied by the "bundle" command, and leverages Markdown files, utilizing `rake` tasks for site preview and generation.

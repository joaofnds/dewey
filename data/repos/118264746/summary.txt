Eclector is a portable and extensible Common Lisp reader library designed for parsing Lisp code. It enables robust error recovery, allowing it to continue reading even with syntax errors, which is beneficial for static analysis. Key features include customizable parse results and the generation of concrete syntax trees, facilitating advanced code manipulation and analysis. This library is built in Common Lisp and targets developers working on Common Lisp tooling, language processing, or projects requiring a flexible and resilient reader implementation. It is a library, providing core functionality for other Common Lisp projects.

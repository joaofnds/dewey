`Audio Router` is a standalone GUI application written in C++ for Windows that enables users to selectively route audio streams from running programs to different audio output devices. It offers functionalities including audio stream routing, audio duplication across multiple devices, and saved routing configurations that persist across application launches. Key features include the ability to route audio from applications, including Metro apps, and a volume mixer integration. This project is a desktop application designed to provide users with granular control over audio output configurations.

MarvellousSuspender is a Chrome extension, written in JavaScript, designed to automatically suspend inactive browser tabs, freeing up system memory and CPU resources to mitigate performance issues and browser crashes. Based on "The Great Suspender" but without tracking, the extension allows users to configure a time interval after which tabs are suspended, effectively acting as a memory management tool. It features a build process using npm, generating a Chrome extension package (CRX format) for installation, and includes a localization component utilizing Crowdin for internationalization. This project type is a browser extension targeting end-users experiencing performance bottlenecks within the Chrome browser.

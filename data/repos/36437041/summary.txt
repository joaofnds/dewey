Artillery is a complete, open-source load testing platform designed for production-grade performance testing of APIs, GraphQL, WebSockets, and more. Built with JavaScript and utilizing Node.js modules, it supports serverless and distributed load testing at scale, including integration with AWS Lambda and Fargate. Key features include Playwright-based browser testing, extensive integrations for monitoring and CI/CD, powerful workload modeling, and extensibility via a plugin API. The platform is primarily a developer productivity tool aimed at software engineers focused on performance and reliability testing, with a target audience of backend developers and DevOps engineers.

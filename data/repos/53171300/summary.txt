Tile38 is a real-time geospatial data store and geofencing server written in Go. It provides a spatial index and supports various object types including points, bounding boxes, GeoJSON, and Geohashes. Key functionalities include spatial search methods (Nearby, Within, Intersects), real-time geofencing via webhooks or pub/sub, and support for multiple protocols like HTTP, WebSockets, and the Redis RESP protocol. This project is a standalone application (server & CLI) and offers a comprehensive set of client libraries across multiple languages, targeting developers working with location-based data and applications.

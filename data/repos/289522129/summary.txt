`Mazes` is a web application for generating mazes, built using Elixir and PhoenixLiveView. It provides a user interface accessible via a web browser at `mazes.angelika.me`. The application leverages Elixir and Phoenix, incorporating front-end dependencies managed with Node.js, to construct and serve the maze generation functionality. This project is a standalone web application demonstrating Elixir and PhoenixLiveView capabilities.

`valtan` is a Common Lisp to JavaScript compiler, enabling the use of Common Lisp features, including macros, for front-end web development. It provides code obfuscation, minification, and source map generation for debugging, facilitating integration with JavaScript libraries and frameworks, including React. The project is in an alpha stage, aiming for ANSI Common Lisp compatibility and utilizing the `sacla` test suite for evaluating its implementation. It is a standalone command-line tool and library aimed at Common Lisp developers seeking to leverage Lisp's features in the browser.

`lume` is a Lua library designed for game development, providing a collection of utility functions. It offers a range of features including mathematical operations (clamp, round, lerp, vector calculations), random number generation, array manipulation (push, remove, sort, filter, unique), string formatting and utility functions, such as memoization, and  string manipulation. The library also includes functions for table operations, like merging, concatenating and inverting tables, as well as a method for calling a lua string to generate a function based on parameters. This is a standalone library intended for use in Lua game development projects.

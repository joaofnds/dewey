Invidious is a libre, alternative front-end to YouTube, providing a lightweight, privacy-respecting web application for video viewing. Built using the Crystal programming language, it offers features such as ad-free video playback, no tracking, and customizable themes, with subscriptions independent of Google. The application allows for importing and exporting subscriptions/history from other platforms like YouTube and NewPipe, and exposes a developer API. It is a standalone web application and provides multiple instances that users can select to use.

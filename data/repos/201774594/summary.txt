`babashka/sci` is a Clojure library providing a configurable Clojure/ClojureScript interpreter suitable for scripting and embedded DSLs. It enables safe evaluation of Clojure and ClojureScript code from user input, supporting the JVM, GraalVM native images, and JavaScript environments.  Key features include API access for evaluation of strings, support for namespaces, macros, vars, reader conditionals, and integration with JavaScript libraries. This library is designed for developers seeking to incorporate Clojure-based scripting or DSL capabilities within their projects and is a library utilized by Babashka, nbb, scittle and others.

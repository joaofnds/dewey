`sol` is a next-generation package manager being developed for the Solus operating system. It aims to replace the existing `eopkg` package manager, focusing on improved update efficiency, stability, and a modern design aligned with the needs of a rolling-release distribution.  Written in C, the project intends to provide core functionalities like containerized package formats, installation and removal operations, and robust repository handling.  This is a standalone application project intended for system administrators and users of the Solus operating system, specifically focusing on improving the user experience of software updates.

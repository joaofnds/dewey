FerretDB is an open-source alternative to MongoDB, designed as a document database solution. It functions as a proxy, translating MongoDB 5.0+ wire protocol queries to SQL and leveraging PostgreSQL with the DocumentDB extension as its backend. Developed in Go, the project allows users to utilize existing MongoDB drivers and tools, providing a drop-in replacement in many scenarios. It is a standalone application, providing database functionality. The target audience includes developers seeking an open-source, compatible database alternative to MongoDB.

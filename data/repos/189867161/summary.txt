`pastel` is a command-line tool written in Rust designed for color manipulation and analysis. It provides functionalities for generating, converting, and analyzing colors, supporting various formats like RGB, HSL, and CIELAB, and offering ANSI 8-bit and 24-bit color representations for terminal display. Key features include color format conversion, color analysis, screen color picking, generation of visually distinct color sets, and text colorization within shell scripts. This standalone application targets developers and terminal users requiring color-related utilities.

`pastel` is a command-line color manipulation tool written in Rust that provides comprehensive color space conversion, analysis, and generation capabilities for developers and designers. It supports multiple color formats including RGB (sRGB), HSL, CIELAB, CIELCh, and ANSI 8-bit/24-bit representations, enabling seamless conversion between different color spaces. The tool addresses common workflows in web development, UI/UX design, and terminal applications by offering features like color picking, palette generation, visual distinctness algorithms, and terminal text colorization. Target users include front-end developers, graphic designers, and system administrators who need programmatic color manipulation for CSS preprocessing, design system automation, shell scripting, and terminal interface development. Its composable CLI architecture allows piping operations for complex color transformations and batch processing workflows.

`before.nvim` is a Neovim plugin designed to facilitate navigation through edit history across multiple buffers. It enables users to jump to previous and next edit locations, similar to `changelist` functionality, using keybindings. The plugin provides features for visualizing the edit history within the quickfix list or via a Telescope picker, enhancing navigation and workflow. Built with Lua and intended for use within the Neovim ecosystem, the plugin allows for configuration of history size and wrapping behavior, and integrates with the Telescope plugin for advanced edit history browsing. This is a Neovim plugin, targeted at Neovim users looking to improve their editing experience.

`uds` is a Python command-line application designed to store files in Google Drive without consuming quota by splitting binary files into base64 encoded text within Google Docs. The application provides functionalities to upload, list, download, and delete files, along with extensions for bulk operations like batch processing and wiping files. It leverages the Google Drive API and requires a client configuration file for authentication. The primary function of this project is a standalone utility for circumventing Google Drive storage limitations, targeting users seeking to store large files.

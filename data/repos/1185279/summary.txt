OpenEmu is a macOS application providing a unified interface for retro video game emulation. It leverages a modular architecture with plugin support for a wide range of game engines, enabling the emulation of numerous retro game consoles.  Built with Swift and utilizing macOS technologies like Cocoa, Metal, and Core Animation, OpenEmu facilitates gameplay through a native macOS user interface, including auto-updating functionality via Sparkle. It functions as a standalone GUI application targeted towards users seeking to play retro video games on macOS.

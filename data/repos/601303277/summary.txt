`sqruff` is a fast and efficient Rust-based SQL linter and formatter designed for ensuring query quality and consistency across various development workflows. Key features include advanced, customizable SQL linting capabilities, automated formatting, and support for multiple SQL dialects such as ANSI SQL (default), BigQuery, Athena, Clickhouse, Databricks, DuckDB, Mysql, PostgreSQL, Redshift, Snowflake, SparkSql, SQLite, Trino, and more. It can be easily integrated into websites, installed using tools like brew or pip, or used as a GitHub Action for CI/CD pipelines. The project is open-source under the Apache License 2.0 and offers a browser-based playground for users to try it out.

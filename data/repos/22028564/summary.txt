`jonathan` is a Common Lisp library providing JSON encoding and decoding functionalities, designed for high performance. Key features include converting Lisp objects to JSON strings and vice-versa, supporting various input and output formats (plist, alist, jsown, hash-table, octets), and offering customizable encoding/decoding through methods like `compile-encoder` and the `with-object` macro. The library facilitates custom object serialization, keyword normalization, and unicode escape sequence handling.  The project type is a library, and it targets Common Lisp developers needing efficient JSON serialization and parsing within their applications.

`hashcat` is a standalone, high-performance password recovery utility designed for cracking password hashes. Written in C, it leverages CPUs, GPUs (including CUDA and OpenCL), and other hardware accelerators for parallelized hash computation and supports over 300 hashing algorithms. Its core functionality enables users to recover passwords from various hash formats using different attack modes, targeting system administrators and security professionals. This project facilitates distributed password cracking and is licensed under the MIT license.

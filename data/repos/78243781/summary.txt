`Super Productivity` is a TypeScript-based, cross-platform (Electron) task management application designed for enhanced productivity and time tracking. Its core functionality includes to-do list management, timeboxing, and time tracking, with integrations for Jira, GitHub, GitLab, and OpenProject. The application features sub-tasks, project and tag organization, break reminders, a Pomodoro timer, and data synchronization via Dropbox and WebDAV, along with keyboard shortcuts and a short-syntax for streamlined task creation. Primarily a standalone desktop application, it provides both a web app and installable packages for various operating systems.

`jsonapi` is an Elixir library providing JSON:API serialization, deserialization, and query handling functionalities. It enables developers to render Elixir data models into JSON:API documents, parse and validate JSON:API query parameters (sorting, filtering, includes, sparse fieldsets, pagination), and enforce JSON:API specification compliance. The library integrates with Phoenix applications via `JSONAPI.View` for simplified view creation and supports customization through field transformation, pagination strategies, and configuration for host, scheme, and namespace. Primarily a library, it targets Elixir developers building APIs adhering to the JSON:API standard.

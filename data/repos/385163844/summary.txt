`Twilight.nvim` is a Lua plugin for Neovim 0.5 designed to enhance code readability by dimming inactive code sections using TreeSitter. It provides functionalities to toggle, enable, and disable the dimming effect, supporting configurable alpha values, color settings, and context lines. The plugin leverages TreeSitter for more accurate dimming and integrates with `zen-mode.nvim` for automatic activation, catering to Neovim users seeking an improved coding experience by reducing visual clutter and emphasizing active code regions.

rqlite is a distributed, fault-tolerant relational database built on SQLite, providing a lightweight and user-friendly solution for data storage and management. It offers a developer-friendly HTTP API, CLI, and client libraries, enabling features such as full-text search, JSON support, and SQLite extensions. This Go-based project supports dynamic clustering, read-only nodes, and automatic backups, making it suitable for both developers and operators requiring a highly available, scalable, and easily deployable database solution, operating within the distributed systems domain.

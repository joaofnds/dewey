`adblock-rust` is a Rust-based adblock engine library designed for content filtering on the web. It provides network blocking, cosmetic filtering, resource replacements, and supports various adblock rule syntaxes, including uBlock Origin and iOS content-blocking formats. Key functionalities include compilation to native code or WebAssembly, with bindings for Rust, JavaScript (NodeJS), and Python. This library facilitates developers in building ad-blocking capabilities into browsers, research tools, and proxies, offering features like CSS validation during rule parsing, content blocking format translation for iOS, and domain resolution options to optimize performance and reduce binary size.

Buttercup Mobile is a React Native mobile application designed for the Buttercup password manager ecosystem, enabling secure password storage and management on iOS and Android platforms. Key features include on-device AES 256-bit encryption, auto-hide on background, vault auto-locking, entry management, offline read-only access, and biometric authentication. The application leverages React Native for cross-platform native development using a TypeScript codebase, alongside Redux and React Native Router Flux for state management and navigation, with native code (Objective-C/Java) utilized for performance-critical cryptographic operations. This is a standalone mobile application providing secure password management capabilities.

`dlv` is a JavaScript library designed for safe and efficient deep property access within nested JavaScript objects. It provides a concise and performant utility function to retrieve values from an object using a dot-notation or array-based key path, handling potential `undefined` values gracefully. The library supports ES Modules, CommonJS, and global usage, and includes an optional default value for missing keys. As a developer productivity tool, `dlv` offers a compact and easily integrated solution to safely navigate object structures, targeting JavaScript developers needing to access nested data.

imgproxy is a standalone, high-performance image processing server designed for resizing, converting, and securing remote images. Built in Go and leveraging the libvips image processing library for speed, it efficiently handles image manipulation tasks such as format conversion (including AVIF and WebP) and resizing.  Its features include URL signature-based security, dimension-based image bomb protection, and support for Docker deployment, targeting developers needing a fast and secure solution for image optimization in web applications and other image-intensive scenarios.  It operates as a microservice and a tool to optimize image serving workflows.

**"gmail-to-sqlite" is a Python application that enables users to index their Gmail account data into a local SQLite database for analysis and archival purposes. Key features include incremental sync, full sync with deletion detection, parallel processing, robust error handling, graceful shutdown, and type safety. The project utilizes the Google Cloud API, OAuth 2.0 credentials, and multi-threading for improved performance. This tool is particularly useful for data scientists, researchers, or anyone seeking to perform vector embedding and clustering on their Gmail data.**

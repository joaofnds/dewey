Sloth is a native macOS application providing a graphical user interface (GUI) for inspecting open files, directories, sockets, pipes, and devices used by running processes. Functionality includes filtering and sorting based on various criteria, viewing socket status, and detailed process information, leveraging the `lsof` command-line tool. Developed in Objective-C using Cocoa, this standalone application targets system administrators and developers needing to analyze system resource usage and process behavior on macOS.

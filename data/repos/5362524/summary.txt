Source Code Pro is an OpenType font family designed specifically for user interface (UI) environments and coding contexts. This repository provides the font source files and build instructions, enabling customization and integration. The project delivers fonts in OpenType (`otf`) and TrueType (`ttf`) formats, including variable font support, and targets developers and designers seeking high-quality, monospaced fonts suitable for UI development and code readability. The project is a font library distributed under an open-source license.

`dsq` is a command-line tool written in Go designed for executing SQL queries against diverse data formats, including JSON, CSV, Excel, Parquet, and others. Its core functionality enables users to transform and analyze data residing in various file types using standard SQL syntax. The tool leverages SQLite via DataStation, incorporating features such as schema dumping, caching, and an interactive REPL environment, enhancing data exploration workflows. As a standalone CLI application, `dsq` targets data analysts, developers, and anyone needing a SQL-based interface for interacting with structured data files.

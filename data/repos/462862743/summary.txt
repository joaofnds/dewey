`oxide.ts` is a TypeScript library implementing Rust's `Option<T>` and `Result<T, E>` types, providing a functional approach to handling nullability and error management. It offers core features including `Option` and `Result` types with methods for mapping, unwrapping, and converting between types, along with advanced features like pattern matching via `match`, and safe operations for handling function outcomes and promises. The library includes zero dependencies, full test coverage, in-editor documentation, and aims to improve developer productivity through enhanced type safety and functional programming paradigms for TypeScript projects, targeting TypeScript developers. It is a developer productivity library.

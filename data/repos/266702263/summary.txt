`rough-notation` is a TypeScript library designed to create and animate hand-drawn style annotations on web pages. Leveraging the RoughJS library, it provides functionalities to add various annotation styles like underlines, boxes, circles, and highlights to HTML elements, with configurable animation, color, stroke width, and padding. The library offers annotation grouping for ordered animations and includes methods for showing, hiding, and removing annotations, along with options for updating styles dynamically. Primarily a frontend library, it's intended for web developers seeking to enhance visual presentation.

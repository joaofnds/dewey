`cowboy` is a small, fast, and modern HTTP server written in Erlang/OTP, designed for high-performance web applications. It provides features for handling HTTP/1.1 and HTTP/2 protocols, including HTTPS, WebSocket server support, and REST API capabilities. As a production-ready web server, it acts as a core component for building Erlang-based web frameworks and services, targeting developers building scalable and concurrent web applications. This project functions as a standalone application.

Steel Bank Common Lisp (SBCL) is a high-performance compiler and runtime system for the Common Lisp programming language. This repository serves as a mirror of its official source code.  It provides a complete implementation of the Common Lisp standard, enabling developers to write and execute Lisp programs.  The project facilitates the development of a wide range of applications, from scientific computing to scripting and general-purpose programming, and it is primarily aimed at developers working with Common Lisp.

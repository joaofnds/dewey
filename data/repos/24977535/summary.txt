`transfer.sh` is a command-line utility written in Go for easy and fast file sharing. It functions as a standalone application providing file upload and download functionality accessible via `curl`. Users can upload files using `curl` with options for encryption, deletion, and setting download limits, while downloads are performed via simple `curl` requests. The project supports various storage providers including Amazon S3, Google Drive, Storj, and local file systems. This is a CLI application aimed at developers and system administrators needing a quick file-sharing solution.

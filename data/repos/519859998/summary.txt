AFFiNE is a privacy-focused, open-source, and local-first knowledge base application designed as a Notion and Miro alternative. Built with TypeScript and utilizing Electron for cross-platform compatibility, it features a merged canvas and document paradigm, enabling users to combine rich text, embedded content, and visual elements. Key functionalities include real-time collaboration, AI integration for content generation, and support for self-hosting, leveraging technologies like CRDTs for data synchronization, and Rust-based components such as Blocksuite and OctoBase. This project represents a standalone desktop application with a web application counterpart, targeting users seeking a customizable and versatile workspace for planning, note-taking, and knowledge management.

`traefik/whoami` is a tiny, standalone Go web server designed to echo information about the server environment and HTTP requests. It provides endpoints to display OS and network information, supports configurable features like port number, name, and verbose logging, and includes functionalities like a health check, WebSocket echo, and data size simulation. Built with Go, the application offers examples of its usage with Docker, illustrating its deployment and configuration for various scenarios, including TLS support. It is intended as a simple utility, potentially for testing or demonstration purposes, and is deployable as a containerized application.

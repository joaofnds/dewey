**Later** is a JavaScript library for defining complex recurring schedules and calculating their future/past occurrences, targeting web developers and system administrators in DevOps and task automation domains. It provides multiple schedule definition methods including Cron expressions, English text parsing, and a chainable API for creating composite schedules with exceptions. The library supports both Node.js and browser environments, enabling developers to build sophisticated scheduling systems for tasks like automated reports, system maintenance, monitoring, and notifications. Key features include extensible time periods, custom modifiers, range-based scheduling for meetings/activities, and automatic resolution handling. It solves the problem of complex schedule calculation beyond simple cron jobs, particularly useful for enterprise applications requiring flexible, human-readable schedule definitions with precise occurrence computation.

`starship/starship` is a cross-shell prompt written in Rust, designed for customization and speed. It provides a highly configurable command-line interface (CLI) prompt, displaying relevant information at a glance and supporting various shells, including bash, fish, zsh, and PowerShell. Key features include its speed, intelligent display of context-specific information, and extensive configuration options. This project is a standalone CLI application aimed at developers and system administrators seeking a performant and customizable shell prompt.

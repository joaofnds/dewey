`stackline` is a Lua-based plugin for macOS, designed to visually enhance the `yabai` tiling window manager. It provides unobtrusive on-screen indicators to represent and manage window stacks within `yabai`. Functionality includes displaying app icons or minimal indicators showing stack positions and window counts, supporting multi-monitor setups, and offering control via shell commands or Hammerspoon. This project is a Hammerspoon plugin, extending the functionality of the macOS window manager `yabai`.

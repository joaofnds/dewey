Compton is a standalone C-language compositor for the X Window System (X11), forking and enhancing the functionality of xcompmgr. It provides advanced compositing features like OpenGL and X Render backends, inactive window transparency and dimming, titlebar and menu transparency, colored shadows, and a configurable fade system. The project is a command-line application, depending on libraries such as libx11, libxcomposite, and libxrender, allowing users to customize the appearance of their X11 desktop environment.

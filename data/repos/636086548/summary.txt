This TypeScript-based project, f1-dash, offers a real-time Formula 1 dashboard that provides comprehensive telemetry and timing data. The dashboard displays leaderboards, tire information, gaps, laps, mini sectors, and more in a web environment. Key features include live updates, interactive visualizations, and customizable views. The technology stack includes Next.js for the frontend, Rust for backend services, and TypeScript throughout. This unofficial F1 dashboard is intended for enthusiasts seeking an immersive, real-time experience of the Formula 1 racing season.

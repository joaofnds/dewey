`Joker` is a standalone command-line application and developer productivity tool providing a Clojure interpreter, linter, and formatter written in Go. It enables scripting, code linting, and formatting for Clojure, ClojureScript, and Joker dialects, with integrations for popular editors like Emacs, Sublime Text, and VSCode. Key features include a REPL, script execution, expression evaluation, and linting/formatting modes, targeting Clojure developers seeking a lightweight and fast-startup scripting environment with tooling for Clojure code. The project leverages Go as its primary language, with a focus on compatibility with Clojure syntax and semantics.

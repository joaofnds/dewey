`helix-editor/helix` is a post-modern, modal text editor inspired by Kakoune and Neovim, written in Rust. It provides Vim-like modal editing, multiple selections, built-in language server support, and smart syntax highlighting via tree-sitter. This standalone, terminal-based application aims to offer an efficient and customizable text editing experience, leveraging Rust's performance capabilities. Its primary domain is developer productivity, specifically targeting users who prefer or are migrating from modal text editors.

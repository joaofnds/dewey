`vim-peekaboo` is a Vim plugin designed to enhance user interaction with registers. It extends normal mode (`"` and `@`) and insert mode (`<CTRL-R>`) functionality to display the contents of Vim registers in a sidebar, aiding in text manipulation and code editing. This plugin provides features for customizing the display window, including a fullscreen toggle, along with options for delaying window opening and compact display modes. Built using Vim script, it targets Vim users seeking improved visibility and control over register contents within their editing workflow, and is distributed under the MIT License.

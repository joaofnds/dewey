`jsonui` is a command-line tool written in Go that provides an interactive JSON explorer within the terminal environment. It allows users to pipe JSON data into the tool and navigate the data interactively, with functionalities to expand/collapse nodes, copy paths, and view help documentation. The application utilizes terminal-based input for navigation and is installable via `go get`. This project is a standalone command-line application designed for developer productivity, enabling efficient exploration and inspection of JSON structures.

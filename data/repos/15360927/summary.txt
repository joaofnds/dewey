`cider-nrepl` is a Clojure library providing nREPL middleware to enhance Clojure development tools, particularly CIDER, within editors like Emacs and Vim. It enables functionality such as definition lookup and code completion for Clojure code. Built using Clojure and leveraging the nREPL protocol, this library is a plugin/extension designed to integrate with existing Clojure development workflows and increase developer productivity.

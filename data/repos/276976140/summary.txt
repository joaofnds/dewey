`mergestat-lite` is a command-line interface (CLI) tool, written in Go, designed for querying git repositories using SQL. It enables users to extract information from git repositories and related data sources by executing SQL queries against the codebase. Key functionalities include generating reports, performing status checks, and analyzing codebases directly, with support for output formats like JSON, CSV, and NDJSON. The tool targets software developers and system administrators needing to analyze codebases and generate reports, offering an alternative to shell scripting for Git repository analysis.

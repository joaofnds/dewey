G2 is a TypeScript visualization grammar library designed for declarative chart creation, targeting data exploration, dashboard building, and storytelling applications. It provides a progressive and extensible API for constructing interactive charts, supporting various chart types through a system of marks, transforms, scales, and coordinates. The library utilizes a declarative API and a powerful rendering engine (G, supporting Canvas, SVG, and WebGL) to offer data-driven animations and interactions. It is a library for frontend developers focused on data visualization and is installable via npm or yarn.

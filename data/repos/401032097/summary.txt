`lemmeknow` is a Rust-based command-line interface (CLI) tool and library designed for rapid identification of text and data, specializing in cybersecurity and cryptography contexts. It utilizes regular expressions (regex) to analyze input, including files and strings, to identify patterns associated with various data types. Functionality includes identifying hardcoded strings from network packets or malware analysis and offers JSON output options. The project is implemented in Rust, providing performance benefits; it's available as a standalone executable and as a library crate, supporting usage in other Rust projects and webassembly compilation.

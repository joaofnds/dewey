ReLaXed is a JavaScript-based PDF generation tool that leverages Chromium's headless browser capabilities to convert HTML/CSS/JavaScript documents into high-quality PDFs. Built for web developers and content creators, it supports modern web technologies including Pug templating, SCSS preprocessing, and integrates with mathematical typesetting via MathJax, data visualization through Vega-Lite and Chart.js, and Markdown parsing. The tool addresses the challenge of creating complex document layouts for reports, presentations, books, and business documents by combining the flexibility of web technologies with PDF output, offering an alternative to traditional LaTeX or proprietary document generation systems. It provides real-time compilation with file watching capabilities, enabling interactive document development workflows.

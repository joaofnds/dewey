`rubocop-emacs` is an Emacs Lisp package providing an interface to RuboCop, a Ruby code style checker. The package facilitates code style checking, auto-formatting, and auto-correcting of Ruby code within the Emacs editor. Functionality includes project-wide, directory-specific, and file-level operations, configurable through Emacs Lisp variables to adjust RuboCop command execution. It integrates with Emacs's `ruby-mode` and provides keybindings for easy access to RuboCop features. This project is an Emacs plugin designed for Ruby developers using Emacs.

This repository provides a Dockerized Apache Airflow environment, enabling workflow management and task scheduling. It offers pre-configured Dockerfiles for various Airflow executors (Sequential, Local, and Celery), facilitating easy deployment and configuration using Docker Compose. The project is written primarily in Shell scripting for Dockerfile setup and leverages Python (3.7-slim-buster) and utilizes PostgreSQL and Redis for backend and queue functionalities, respectively, streamlining data pipeline orchestration for data engineers and DevOps teams. The project is a Docker image and configuration files, serving as a ready-to-use solution for running and managing Airflow workflows within a containerized environment.

`sonarlint-core` is a Java library providing the core functionality for SonarLint analysis, a developer productivity tool designed to promote Clean Code practices. It's used by SonarLint plugins for IDEs like Eclipse and IntelliJ, as well as the SonarLint language server used by VSCode. The library enables static code analysis to identify code quality issues. This library is a core component of the SonarLint ecosystem, facilitating static analysis within various development environments and improving code quality.

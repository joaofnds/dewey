Excalidraw is a TypeScript-based, open-source web application and npm package facilitating collaborative, hand-drawn-style diagram creation. The project enables users to sketch diagrams, wireframes, and other visual elements on an infinite canvas, supporting features like image support, shape libraries, export to various formats (PNG, SVG), and real-time collaboration with end-to-end encryption. Built for frontend developers integrating diagramming capabilities, the application features PWA support, local-first data persistence, and shareable links.

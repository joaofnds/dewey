`lispunion/code-formatter` is a Scheme code formatting library and command-line tool designed to automatically format Scheme source code, enhancing code readability and style consistency. It supports in-place file editing and provides features like line break and indentation control, along with alphabetical sorting of function definitions where semantically safe. Built using Chicken Scheme 5.x, the project is a standalone command-line application and aims to align formatting with the scheme-style style guide. The primary users would be Scheme developers seeking to maintain standardized code formatting.

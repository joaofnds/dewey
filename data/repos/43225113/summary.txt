`yq` is a command-line utility, written in Go, designed for processing YAML, JSON, XML, CSV, TOML, and properties files. It utilizes a `jq`-like syntax for querying and manipulating data within these formats, enabling users to read, update, and merge data, as well as convert between different formats. Key functionalities include in-place file updates, integration with environment variables, and support for complex expressions. It functions as a standalone CLI application, suitable for DevOps tasks, configuration management, and data transformation, targeting system administrators and developers.

Unison is a statically-typed, functional programming language designed for building distributed systems with a focus on content-addressed code, eliminating traditional build steps. This Haskell-based project features perfect incremental compilation, instant renaming of definitions, and semantically-aware version control, enhancing developer productivity. Key functionalities include a codebase server and a Language Server Protocol (LSP) implementation, facilitating development and integration with tools. It is a standalone programming language, intended for developers interested in building and deploying efficient and robust applications.

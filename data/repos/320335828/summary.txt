This repository is a curated collection of wordlists specifically designed for cybersecurity professionals and penetration testers. It serves the information security domain by providing organized dictionaries for various attack vectors including directory enumeration, password brute-forcing, username discovery, and web application security testing. The wordlists are categorized by content type (discovery, passwords, usernames, vulnerabilities) and include specialized lists for platforms like HackTheBox, Apache servers, SQL injection, LFI/RFI attacks, and common web technologies like Joomla and JSP. Available as Docker images across Alpine, Debian, and Ubuntu distributions, it targets security researchers, ethical hackers, and penetration testers who need comprehensive word dictionaries for automated security assessment tools like dirb, gobuster, hydra, and similar enumeration frameworks used in vulnerability assessment and penetration testing workflows.

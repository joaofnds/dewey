Retina is an eBPF-based Kubernetes network observability platform, providing centralized monitoring of application health, network health, and security.  It collects customizable telemetry and exports it to various storage backends, offering actionable Prometheus metrics and streamlined packet captures.  Built in Go, the project utilizes eBPF for network traffic inspection and provides a CLI for capturing and analyzing network data within Kubernetes clusters.  This is a standalone application with a focus on providing network insights for Kubernetes cluster administrators and DevOps engineers.

`particles.js` is a lightweight JavaScript library designed for creating customizable particle animations for web applications. This library enables the dynamic generation and manipulation of particles with configurable properties such as color, shape, size, and movement behavior, including interactivity features like mouse-based repulsion and attraction. Users can load the library and configure particle effects via a JSON configuration file or by using a generator.  It is intended for frontend developers looking to enhance the visual appeal and interactivity of their web projects using a JavaScript-based solution.

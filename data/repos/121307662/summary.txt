`Recursive` is a Python-based project for creating the Recursive Sans & Mono variable font family designed for code and UI applications. It leverages `fontmake` and `fonttools` for font building and variable font generation, integrating with Google Fonts. The project provides a complete font development workflow, including source files, build scripts, and documentation for creating, testing, and releasing variable and static font variations across multiple language supports. It is a font project, offering a free software license and targeting type designers and developers.

RabbitMQ Server is an open-source, multi-protocol messaging and streaming broker designed for robust inter-application communication. It supports AMQP 1.0, AMQP 0-9-1, MQTT, STOMP, and RabbitMQ Stream Protocol, enabling diverse client connectivity.  The project provides core server functionality, built-in plugins, comprehensive documentation, and Kubernetes operator support for deployment. This standalone server application, primarily implemented in Erlang, aims to facilitate reliable message exchange and event streaming for distributed systems and is targeted towards developers and system administrators.

`moveable` is a TypeScript library providing a comprehensive suite of interactive manipulation capabilities for web elements. It empowers developers with features like draggable, resizable, scalable, rotatable, warpable, pinchable, groupable, and snappable functionalities. The library supports various JavaScript frameworks including React, Preact, Angular, Vue, Svelte, and Lit, enabling seamless integration across diverse frontend architectures. This project functions as a JavaScript library intended for frontend developers, streamlining the implementation of dynamic and interactive user interface elements within web applications.

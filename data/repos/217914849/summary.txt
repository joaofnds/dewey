Gleam's `stdlib` is the standard library for the Gleam programming language. It provides core functionalities and types for Gleam projects, supporting both Erlang and JavaScript targets. This library offers essential building blocks and utilities, compatible with Erlang/OTP 26+ and modern JavaScript runtimes, enabling Gleam developers to create portable and performant applications. As a library, it is integrated into Gleam projects via a package manager and provides fundamental features for Gleam development.

JS Paint is a web-based, pixel-perfect remake of the classic MS Paint application, built using JavaScript and HTML5 canvas. This standalone web application enables users to perform raster image editing and manipulation, including drawing, image editing, and color palette management, while also offering features like theme customization, accessibility tools (e.g., dwell clicker, speech recognition), and integration with online services like Imgur. The project supports various image and color palette file formats. Targeted for anyone who wants to use paint on the web.

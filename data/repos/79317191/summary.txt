Typesense is a fast, open-source search engine alternative to Algolia and a more user-friendly alternative to Elasticsearch. Built in C++, it provides typo-tolerant, in-memory fuzzy search capabilities for building performant search experiences. Key features include faceting, filtering, geo-search, vector search for similarity search, and hybrid search capabilities using embeddings. This standalone application is designed to be easily integrated, operated, and scaled, with API clients available in multiple languages.

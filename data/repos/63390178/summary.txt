`capitaine-cursors` is an x-cursor theme for GNU/Linux and Windows, inspired by macOS and built upon KDE Breeze. It provides a visually consistent cursor experience designed to complement the author's `La Capitaine` icon pack. The theme comprises SVG-based cursor icons designed to scale across various pixel densities (DPIs) and offers light and dark variants. Functionality includes building from source using `inkscape` and `xcursorgen`, with installation instructions provided for various Linux distributions and Windows, facilitated by pre-built binaries and package managers, making it a standalone desktop theming package aimed at end-users of desktop environments like KDE and GNOME.

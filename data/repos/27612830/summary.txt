`evil-snipe` is an Emacs package providing two-character text searching functionality, emulating `vim-sneak` and `vim-seek` within `evil-mode`. This Emacs Lisp plugin enhances text navigation by allowing users to jump quickly to locations based on two-character sequences, with incremental highlighting of candidate matches. Key features include global and local mode activation, custom keybindings, search scope customization, character aliasing, and integration with other plugins. It is a developer productivity tool specifically designed as an extension for Emacs users employing the `evil-mode` package for Vim-like editing.

Rectangle is a standalone macOS application written in Swift, designed to enhance window management capabilities. Based on Spectacle, it provides keyboard shortcuts and snap areas for moving and resizing windows. Key functionalities include window snapping to screen edges and corners, maximizing windows, and ignoring specific applications to prevent shortcut conflicts. The application utilizes MASShortcut for keyboard shortcut recording and stores configuration via NSUserDefaults. It's targeted toward macOS users seeking improved window organization and efficiency.

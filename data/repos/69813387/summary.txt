`Fusuma` is a Ruby application designed for multitouch gesture recognition on Linux systems, enabling intuitive touchpad interactions. It leverages the `libinput` driver to interpret gestures such as swipes, pinches, and holds, allowing users to customize actions through a YAML configuration file. The software provides a plugin system for extending functionality, along with features for adjusting sensitivity and intervals. It's a standalone command-line application targeting Linux users seeking enhanced touchpad control.

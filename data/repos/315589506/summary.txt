`Sweep` is a keyboard project based on the Ferris design, focusing on creating customizable split keyboards using promicro-compatible microcontrollers. It offers various PCB designs supporting features like hotswap sockets, MX and Choc switches, reversible PCBs, and Bluetooth connectivity via Nice!Nano. The project provides component lists, build guides, and firmware integrations with QMK and ZMK, making it a hardware design and open-source project targeted at keyboard enthusiasts.

`cljfmt` is a Clojure code formatter designed to detect and automatically fix formatting errors, aligning with Clojure style guidelines while offering customization options. This standalone command-line tool and library, also integrated as a plugin for Leiningen and Clojure CLI, enables users to check and fix Clojure code formatting within projects, supporting various editor integrations. Its core functionality includes indentation correction, whitespace management, namespace sorting, and configuration via `.cljfmt.edn` or project files. This project is a developer productivity tool specifically for Clojure developers.

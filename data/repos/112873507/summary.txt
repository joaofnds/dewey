rate.sx is a command-line cryptocurrency and currency exchange rate service implemented in Python that provides real-time market data through a curl-based HTTP interface. Targeting cryptocurrency traders, developers, and financial analysts, it serves as a terminal-accessible alternative to web-based crypto market platforms. The service offers real-time exchange rates for 500+ cryptocurrencies and 32 fiat currencies, historical price data visualization with ASCII charts, currency conversion calculations, and market capitalization data for top cryptocurrencies. Users can query exchange rates, perform currency conversions, and view historical price trends directly from the terminal using curl commands with flexible time interval specifications and currency pair combinations. The tool solves the need for quick, scriptable access to cryptocurrency market data without requiring GUI applications or API keys, making it particularly valuable for system administrators, traders, and developers integrating crypto data into CLI workflows or automation scripts.

`katana` is a next-generation, Go-based command-line interface (CLI) framework designed for web crawling and spidering. Its core functionality includes both standard and headless crawling modes, JavaScript parsing, customizable form filling, and comprehensive scope control. Key features encompass flexible input options (STDIN, URL, list), various output formats (STDOUT, FILE, JSONL), and the ability to filter and match URLs based on regular expressions and DSL-based conditions.  This standalone application targets security researchers and penetration testers needing a flexible tool for web application reconnaissance and vulnerability discovery.

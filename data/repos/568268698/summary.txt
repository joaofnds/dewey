`Clipboard` is a cross-platform, command-line clipboard manager written in C++ that enhances developer productivity. It provides robust clipboard functionality, including saving and managing multiple clipboards with unlimited history, and supports various data types. Key features include file and directory handling, a scriptable API, and integration with existing clipboard systems, all accessible via a CLI. It is a standalone, terminal application targeting developers and system administrators seeking an efficient and feature-rich clipboard management solution, with pre-built packages available on several package managers including Homebrew, Conda-forge, and Snap.

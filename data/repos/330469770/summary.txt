`what-anime-cli` is a Go-based command-line interface (CLI) application designed to identify anime scenes from images. Functionality includes retrieving anime information from local image files or online image links, leveraging the `trace.moe` service.  The application supports common image formats and provides a straightforward installation via `go install`.  This is a standalone CLI application, built with Go, aimed at anime enthusiasts and users seeking a terminal-based tool for anime scene identification.

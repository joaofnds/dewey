This repository, `vscode-recipes`, provides a collection of practical examples and guides for debugging various technologies within the Visual Studio Code (VS Code) environment. It focuses on enabling efficient debugging of applications and tests across diverse technology stacks, including MERN, Angular, Next.js, Vue.js, Node.js, Ruby on Rails, PHP, and Python, among others. The recipes offer configurations and techniques for debugging specific application types and build processes, and support for runtimes like Electron. It functions as a resource of practical examples and guides for developers using VS Code.

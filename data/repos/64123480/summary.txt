Pharo is the source code repository for the Pharo programming language, a dynamic, reflective, and pure object-oriented language inspired by Smalltalk, and designed for live programming environments. The repository provides the core language implementation and facilitates building Pharo images, including bootstrapping from source code using provided scripts and Dockerfile examples. It supports features critical for its environment, including build automation using Jenkins and export in the Tonel file format. This repository is a core component of the Pharo ecosystem, offering a language and associated tools for developers focused on object-oriented software development.

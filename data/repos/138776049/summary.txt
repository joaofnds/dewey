Sorbet is a fast, powerful type checker designed for the Ruby programming language, aimed at improving developer productivity and code maintainability within Ruby codebases. It provides gradual typing capabilities, allowing incremental adoption in existing projects.  The project offers features including error detection, type inference, and integration with existing Ruby tooling and editors.  This is a developer productivity tool, with significant reliance on Ruby, and Bazel for builds, and targets Ruby developers seeking improved code quality and easier large-scale Ruby project management.

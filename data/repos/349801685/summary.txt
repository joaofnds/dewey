`m11` is a TypeScript-based in-browser assembler and emulator for the PDP-11 architecture. It provides functionalities for assembling PDP-11 assembly source code, including error localization and a comprehensive symbol table export. The project features an emulator with capabilities for accessing processor state, tracking memory and register access, and integrating EMT extensions for OS call faking. This library project targets developers needing to work with or understand PDP-11 assembly and emulation, potentially for educational purposes or as part of related development projects.

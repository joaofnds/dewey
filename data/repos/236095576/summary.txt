Backstage is an open-source framework built in TypeScript for constructing developer portals. Its core function is to streamline developer workflows by unifying infrastructure tooling, services, and documentation within a centralized software catalog. Key features include a software catalog for managing microservices, software templates for project creation, and TechDocs for technical documentation. This project is a web-based framework and plugin ecosystem, primarily targeting software engineers and DevOps teams focused on improving developer experience and managing microservice-based architectures. The project is CNCF Incubation level.

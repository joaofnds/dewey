`jet` is a command-line interface (CLI) tool written in Clojure for transforming data between JSON, EDN, YAML, and Transit formats. It provides functionalities including format conversion, data transformation via Clojure functions specified via command-line arguments or file paths, keywordization of JSON/YAML keys, and streaming support. Built as a GraalVM native image, `jet` prioritizes fast startup times suitable for shell scripting and integrates with JVM environments via Leiningen and `deps.edn`. It is designed for developers seeking a versatile CLI for data format conversion and transformation within a Clojure environment.

PKHeX is a comprehensive C# Windows Forms application that serves as a save file editor for Nintendo Pokémon games, targeting the gaming/Nintendo hacking community. Built on .NET 9.0 with C# 13, it provides binary file parsing and manipulation capabilities for Pokémon save data across multiple console generations, including GameCube memory cards, 3DS saves, and individual Pokémon entity files. The application functions as a hex editor with game-specific logic, offering save file validation, format conversion between generations, Mystery Gift handling, and integration with Pokémon Showdown battle simulator. It addresses the need for technical save file manipulation among Pokémon enthusiasts, ROM hackers, and competitive players who require precise control over game data structures and cross-generational Pokémon transfers.

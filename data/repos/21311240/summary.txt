PKHeX is a C# standalone application designed as a save file editor for the Pokémon core series games. It provides functionalities to view, edit, and save Pokémon data within various save file formats, including .sav, .pk*, and more, and allows for conversion between generations. Key features include importing and exporting Pokémon Showdown sets and QR codes, along with data transfer across different formats and generations. The application utilizes .NET 9.0 and integrates dependencies like QRCoder for QR code generation and pokesprite for sprite resources, indicating a Windows Forms GUI application.

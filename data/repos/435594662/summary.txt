`nyoom.nvim` is a Neovim framework designed to provide a highly customizable and opinionated user experience, functioning as a "doom emacs alternative." Built primarily with Fennel, it leverages the Lua scripting language for configuration and extension. Its core functionality revolves around providing a structured environment for Neovim setup and customization, including features like dotfile management, color scheme application through the `oxocarbon` theme, and comprehensive configurations. This project serves as a personal configuration and setup, falling under the category of dotfiles and a user configuration framework, specifically targeted toward Neovim users.

Work Sans is an open-source grotesque sans-serif typeface family designed for web and digital typography applications. The repository contains font files in multiple formats (TTF, WOFF, WOFF2) and weights, distributed through Google Fonts for web developers and designers. Built with OpenType features and variable font technology, it supports extensive Latin character sets including Vietnamese and over 200 additional languages through the Google Latin Expert glyph set. The typeface targets web developers, UI/UX designers, and graphic designers who need a versatile, professionally-crafted sans-serif font for websites, applications, and digital interfaces. The project utilizes modern font production workflows and provides comprehensive language support for international web projects, solving the need for accessible, high-quality typography in digital design systems.

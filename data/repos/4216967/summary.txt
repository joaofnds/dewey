StumpWM is a highly customizable, keyboard-driven tiling window manager written in Common Lisp, designed for a productive and hackable desktop experience. Functionality centers on managing windows with a structure similar to Emacs buffers, providing features such as window tiling, group management, and extensive customization options through Lisp code. The project leverages Common Lisp (SBCL) and dependencies including clx, cl-ppcre, and alexandria. It is a standalone application targeting users seeking a highly configurable alternative to conventional graphical user interfaces.

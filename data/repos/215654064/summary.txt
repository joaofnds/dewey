Temporal is a distributed, Go-based platform designed for durable execution and workflow automation. This repository provides the source code for the Temporal server, enabling the development of scalable applications by executing application logic (Workflows) resiliently, handling failures and retries automatically. Key functionalities include workflow orchestration, distributed task scheduling, and microservice coordination, with features accessible via CLI and a web UI. It targets developers building robust, distributed systems using Go and provides the core component for building resilient applications.

eqMac is a macOS system-wide audio equalizer and volume mixer application. It provides features like system audio processing, volume boosting, HDMI volume support, and a multi-band equalizer with customizable presets and integration with AutoEQ for headphone equalization. Built primarily with Swift, the application features a native backend for audio processing, a web-based Angular user interface for remote updates, and a system audio driver based on Apple's Null Audio Server Driver Plug-in written in Swift. This standalone GUI application targets macOS users seeking to enhance their audio experience with system-wide control and advanced sound processing capabilities.

`SkyRocket.spoon` is a Lua-based Hammerspoon extension designed to enhance window management on macOS by providing functionality to resize and move windows using mouse interactions combined with modifier keys. It emulates features of tools like Zooom/2 and BetterTouchTool. Key functionalities include enabling users to resize and move windows by clicking and dragging while holding specified modifier keys, with options to customize modifier combinations, mouse buttons, and disable behavior for specific applications. This is a plugin/extension for Hammerspoon, a desktop automation tool, intended for macOS users seeking improved window manipulation capabilities.

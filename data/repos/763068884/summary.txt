`yazi.nvim` is a Neovim plugin facilitating file management through the terminal-based file manager, yazi. It enables users to open yazi within Neovim in a floating window, with features like keybinding customization and integration with other plugins for actions such as searching, replacing text, and copying file paths. Written in Lua, the plugin synchronizes file operations between yazi and Neovim buffers, including LSP server integration for renaming, moving, and deleting files. This plugin is designed to enhance developer productivity by providing an integrated file management experience within the Neovim environment.

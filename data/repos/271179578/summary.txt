`nvim-compe` is a deprecated Neovim auto-completion plugin written in Lua. It provides enhanced code completion functionalities, including support for LSP features like snippet expansion, fuzzy matching, and integration with various sources such as buffers, paths, LSP servers, and external plugins like `vim-vsnip` and `nvim-treesitter`. The plugin offers a customizable API for defining custom completion sources and supports features such as VSCode-compatible expansion handling. This is a Neovim plugin designed for developers using the Neovim text editor to enhance their coding productivity.

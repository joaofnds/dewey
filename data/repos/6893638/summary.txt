Gruvbox is a Vim color scheme designed to enhance the coding experience with a retro-inspired, visually pleasing, and easily distinguishable color palette. This Vim plugin provides dark and light mode options with customizable contrast, supporting extensive filetype highlighting for various languages and plugins like Airline and Lightline. It offers users a range of customization options for a tailored aesthetic and integrates with popular Vim utilities for improved code readability and workflow efficiency. This project is a Vim plugin targeting Vim users who seek a visually appealing and highly configurable color scheme.

Busted is a Lua unit testing framework designed for ease of use, supporting Lua 5.1+, LuaJIT 2.0+, and MoonScript. It provides an extensible assertion library with method chaining and a modular output library for various formats, including pretty terminal output, JSON, and TAP.  The framework facilitates test specification with nested blocks, tags, and mock/spy functionalities for functional testing. This library is intended for Lua developers seeking a streamlined solution for testing their Lua code, including CI/CD integration via TAP output.

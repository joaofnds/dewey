Xit is a standalone, graphical Git client application for macOS, written in Swift. It provides a visual interface for managing Git repositories, offering features such as a sidebar for browsing branches, remotes, and tags, a history list with commit graph visualization, and file lists with diff, blame, and QuickLook previews. Utilizing the `libgit2` library for Git operations, Xit aims for stability and scalability, handling large repositories effectively. The application targets developers who need a GUI-based Git workflow on macOS.

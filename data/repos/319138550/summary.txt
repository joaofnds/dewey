This Go library, `cli/oauth`, provides OAuth authorization flows for Go client applications, primarily for interacting with GitHub. It implements both the OAuth Device Authorization Grant ("Device flow") and a fallback OAuth web application flow leveraging a local web server, enabling authorization in CLI applications and environments without browser redirects. The library offers functionalities for device flow authorization, manual OAuth web application flow, and integrates with GitHub's OAuth services, with the core goal of simplifying OAuth authentication. The project type is a library, targeting Go developers building command-line tools and other client applications that need to authenticate with OAuth providers.

`monolith` is a command-line interface (CLI) tool and Rust library designed to archive complete web pages as single, self-contained HTML files. It embeds all associated CSS, images, and JavaScript assets as data URLs, enabling offline viewing and easy sharing of web content.  Key functionalities include customizable options for excluding or including specific content types, domain whitelisting/blacklisting for asset retrieval, MHTML output support, and integration with services like Apify for cloud-based usage. This standalone application targets users seeking to preserve and share web content, including those interested in web archiving or data hoarding.

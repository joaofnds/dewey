This repository provides a collection of Java-based examples and demonstrations showcasing the Confluent Platform for event stream processing with Apache Kafka, Apache Flink and related technologies. It features a variety of demos, including end-to-end data pipelines, stream processing applications using ksqlDB, and integrations with Confluent Cloud. Core functionalities include examples for connectors (JDBC, cloud connectors), schema registry integration with Avro, microservices architectures, and Kubernetes deployments. The project is primarily comprised of example projects and tutorials designed to help developers learn and implement event streaming solutions within the Confluent ecosystem.

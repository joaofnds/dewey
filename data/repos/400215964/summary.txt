This open-source repository, named novuhq/novu, offers a unified API for sending notifications across multiple channels such as Inbox/In-App, Email, SMS, Push, and Chat. The core features include an embeddable Inbox component with real-time support, a digest engine to combine multiple notifications into a single email, a no-code block editor for emails, and a notification workflow engine. Built using TypeScript, it provides developers with a simplified approach to complex multi-channel delivery, making it ideal for modern products. The project is community-driven and offers various integrations like React, Email, SMS, Push, Chat, In-App, and others.

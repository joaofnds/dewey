`git-bug` is a standalone, distributed, offline-first issue management tool embedded within a Git repository, enabling versioned issue tracking without external dependencies. This Go-based application allows users to manage issues, comments, and user data directly within their Git repositories, facilitating offline work and seamless synchronization across multiple remotes. Key features include native Git storage for issues, distributed version control, fast issue listing and searching, third-party bridges to platforms like GitHub, and flexible CLI, TUI, and web browser interfaces. It is a command-line application designed as a developer productivity tool.

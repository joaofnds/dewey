Oni is a modern modal text editor built on Neovim, designed to enhance developer productivity by combining the power of Vim's modal editing with modern IDE features. Written in TypeScript, it leverages React, Redux, and RxJS for its GUI, providing functionalities such as code completion, syntax highlighting, fuzzy finding, and an embedded browser. This cross-platform application aims to deliver a full-fledged Vim experience with a modern user interface and extensibility through JavaScript-based plugins, targeting software developers seeking a feature-rich and efficient text editing environment.

`Docker-OSX` provides a shell-scripted, Docker-based solution for running macOS virtual machines with near-native performance within a Docker container, targeting OS X security research and development.  It enables X11 forwarding for graphical user interface access and supports features like iPhone USB passthrough via usbfluxd and shared folder functionality. The project leverages QEMU and KVM for virtualization, utilizing shell scripting to manage the macOS environment and Docker for containerization; it further supports various macOS versions and offers a Kubernetes Helm chart for deployment. This is a standalone application designed for system administrators, security researchers, and developers.

`cljstyle` is a Clojure code formatting tool designed to automatically improve code style and readability. It reformats messy Clojure code, addressing issues like indentation, spacing, and overall code structure. Key functionalities include checking code formatting, fixing formatting issues, and offering a pipe command for integration. It supports configuration via `.cljstyle` files and integrates with various tools. This is a standalone command-line application targeting Clojure developers and projects.

Numbr is a web-based application and Chrome extension designed for integrating numerical calculations and conversions directly into a user's browsing and note-taking workflows. Built with TypeScript, it provides features including arithmetic operations, currency conversion with support for various symbols and ISO codes, variable assignment, and summing of numerical data within text. The application enables users to perform calculations on the fly within text-based content, facilitating easy sharing of results via shareable links, making it a developer productivity tool targeted for general users.

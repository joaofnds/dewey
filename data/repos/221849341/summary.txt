`asynq` is a Go library providing a distributed task queue for asynchronous task processing. It leverages Redis for persistence and offers features including task scheduling, retries, priority queues, and deduplication, ensuring at-least-once execution. This library integrates with Prometheus for monitoring, provides a web UI for task and queue management, and includes a command-line tool for inspection and control. It is a backend library designed for Go developers to implement background jobs and improve application scalability.

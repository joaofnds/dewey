FightPandemics is a JavaScript-based web application designed to facilitate community resilience during pandemics by connecting individuals with relevant information and coordinating responses. The project leverages a modern stack including React Hooks for the frontend, Fastify for the backend, and MongoDB for data storage, with Docker for deployment and AWS infrastructure. Key functionalities involve information aggregation, community coordination tools, and potentially geolocation features using the Google Maps API. It's a full-stack web application targeting both frontend and backend developers and community organizers, built to provide a platform for pandemic-related aid and information.

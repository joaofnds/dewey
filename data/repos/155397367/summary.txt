Brooklyn is a macOS screensaver application written in Swift that recreates the animated visual effects displayed during Apple's October 30, 2018 special event at the Brooklyn Academy of Music. Targeting macOS desktop users and Apple enthusiasts, the screensaver operates offline and provides customizable animation preferences including light/dark themes, animation selection, loop controls, and randomized playback order. The application integrates with macOS's native screensaver framework, supports installation via Homebrew package manager, and includes system-level preference configuration. It addresses the niche use case of users wanting to replicate Apple's distinctive event aesthetics on their desktop environment, requiring macOS El Capitan (10.11) or higher and utilizing Core Animation technologies for smooth visual rendering.

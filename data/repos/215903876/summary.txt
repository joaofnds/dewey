`nvim-colorizer.lua` is a Lua-based Neovim plugin designed to provide high-performance color highlighting for code within the editor, specifically focusing on speed and efficiency without external dependencies. The plugin parses and highlights color codes like hex codes and CSS color names and functions. It offers both foreground and background highlighting modes, customizable per filetype and buffer, and includes commands for attaching, detaching, and reloading highlighting. It is a plugin for Neovim users seeking enhanced code readability and color visualization directly within their editor.

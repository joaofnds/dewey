Kulfon is a JavaScript static site generator designed for creating websites using Org Mode, Markdown, and Nunjucks templates. The one-command tool compiles data and templates into HTML pages, similar to Hugo, with support for Sass and Rollup for optimized performance and smaller website sizes. This project is a command-line application aimed at frontend developers seeking a flexible static site generation solution, offering features like built-in support for Org Mode and Markdown, along with options for dependency management and HTTP/2 readiness.

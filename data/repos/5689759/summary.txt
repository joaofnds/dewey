`day-of-datomic` is a Clojure-based example project providing tutorials and sample code for learning and interacting with Datomic, a database. This repository focuses on demonstrating Datomic functionalities through Clojure REPL-based tutorials, covering topics like hello world, social news, provenance, and data functions. Users can evaluate code examples directly within a REPL environment to explore and understand Datomic's features. It serves as an example project intended to educate developers on Datomic usage and related functionalities.

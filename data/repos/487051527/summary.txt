`solarized.nvim` is a Neovim plugin providing a port of the Solarized color palette, a sixteen-color scheme designed for terminal and GUI applications. Written in Lua, the plugin allows users to apply and customize the Solarized theme within Neovim, with features including Treesitter and semantic highlighting support, configurable styles, highlights, and colors, and integration with various Neovim plugins like `lualine` and `barbecue`. This project is a Neovim plugin (a colorscheme and theme) that enables users to apply and customize the Solarized color scheme within Neovim, targeting Neovim users and developers.

This repository provides a Neovim colorscheme plugin implementing the Solarized color palette, targeting software developers and text editor users in the development tools domain. Built in Lua for Neovim v0.9.1+, it features comprehensive syntax highlighting support for Treesitter and LSP semantic highlighting, along with extensive plugin integrations including Telescope, NeoTree, Lualine, and 30+ other popular Neovim plugins. The plugin offers four seasonal variants (spring, summer, autumn, winter), transparency controls, customizable highlight groups and color palettes, and includes both the original Solarized and Selenized color schemes. It solves the problem of consistent, eye-friendly syntax highlighting across the entire Neovim ecosystem while providing developers with granular customization options for their coding environment aesthetics and readability preferences.

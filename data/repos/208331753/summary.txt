esmBot is a TypeScript-based Discord bot designed for server entertainment, offering image manipulation, music playback, and utility commands. Utilizing the Oceanic library, it leverages libvips for efficient image processing and supports features like a local web server for large image uploads, a music playback system powered by Lavalink, and a flexible command handler.  It supports both slash and prefix-based commands and utilizes multiple database backends (PostgreSQL, SQLite). This standalone application targets Discord server administrators and users seeking to enhance their server's functionality and enjoyment.

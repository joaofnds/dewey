`Ahoy` is a Ruby library providing simple, first-party analytics for Rails applications, tracking visits and events across web, JavaScript, and native app environments. It stores data within the application's database by default but is designed to support custom data stores. Core features include visit tracking with comprehensive metadata (traffic source, location, technology), event logging with custom properties, and integration with models via the `visitable` method, as well as GDPR compliance functionalities. This is a Ruby library, intended for Rails developers to gain insights into user behavior within their applications.

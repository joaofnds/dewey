`jj` is a command-line utility written in Go designed for efficient JSON stream editing. It enables users to retrieve and update values within JSON documents using a path syntax, offering functionalities such as value extraction, modification, and deletion. Utilizing GJSON and SJSON under the hood, `jj` prioritizes performance by avoiding unnecessary parsing and can also format and compress JSON output. This standalone CLI application is intended for developers and system administrators who need a fast and straightforward tool for manipulating JSON data in scripts or the command line.

`exordium` is an Emacs configuration designed to enhance C++ and Lisp/Clojure development with IDE-level features, functioning as a customizable "hacker kit" rather than a starter kit. It provides project management via Projectile, Git integration with Magit, and C++-specific utilities including code formatting for the BDE style and integration with the `include-what-you-use` tool. Built upon Emacs Lisp and utilizing packages like Helm, Avy, and Treemacs, `exordium` is an Emacs extension project that aims to improve developer productivity with various features for code completion, navigation, and project management.

`Talltale` is a Clojure and ClojureScript library designed to generate fake data for software testing and prototyping. It provides a collection of functions for generating various data types, including addresses, company details, and personal information, with locale support. The library integrates with `clojure.spec` and `test.check` by offering generators for property-based testing, facilitating the creation of robust test cases. This is a library intended for use by Clojure and ClojureScript developers to improve test data creation.

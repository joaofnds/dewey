Hertz is a high-performance and extensible Go HTTP framework designed for building microservices. It provides features including high usability, robust extensibility, and multi-protocol support (HTTP/1.1 and ALPN). Built on the high-performance network library Netpoll and inspired by other Go frameworks, Hertz offers a layered design enabling custom protocol logic and network layer switching. This framework is a library primarily targeting Go developers focused on building efficient and customizable microservices.

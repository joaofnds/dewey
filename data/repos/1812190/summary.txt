`blackfriday` is a Go library providing a Markdown processor with HTML output generation. It focuses on parsing and rendering Markdown text, supporting features such as tables, fenced code blocks, footnotes, and smart punctuation substitutions, with strong emphasis on input safety. The library offers both v1 and v2 APIs, with v2 introducing an abstract syntax tree for parsing and improved API design. Functionality includes the ability to customize rendering options and integrate with HTML sanitization tools like Bluemonday. `Blackfriday` is a library intended for developers requiring Markdown processing capabilities within Go applications and projects.

`samber/do` is a Go library providing a typesafe dependency injection (DI) toolkit, designed to simplify service management and lifecycle control within Go applications. It enables service registration, invocation, health checks, and graceful shutdown, incorporating features like dependency graph resolution, service override capabilities, and lifecycle hooks. Built upon Go 1.18+ generics, the library is a lightweight, no-dependency solution, suitable for managing dependencies and service interactions, operating as a utility library for Go developers building applications.

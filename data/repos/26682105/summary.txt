Dark Reader is a browser extension, developed in TypeScript, designed to enhance web browsing accessibility and reduce eyestrain by applying a customizable dark mode to web pages. This open-source project analyzes web content and dynamically inverts colors and adjusts brightness/contrast/sepia to generate a dark theme, supporting Chrome, Firefox, Safari, Edge, and Thunderbird. The extension offers an API for developers to integrate dark mode into their websites and includes site-specific fixes. The project is a browser extension targeting users seeking improved readability and eye comfort while browsing the web.

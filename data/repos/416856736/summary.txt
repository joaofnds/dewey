`add-to-calendar-button` is a JavaScript web component library designed to facilitate the creation of customizable "Add to Calendar" buttons for websites and applications. This library enables users to add events to various calendar services, including Google Calendar, Apple Calendar (via iCal), Microsoft 365/Outlook/Teams, and Yahoo. The component supports event configuration with options for timed and all-day events, recurring events, and time zone management, and is compatible with frameworks like Angular, React, and Vue, as well as standard JavaScript. It is a free, standalone library that provides a simple HTML-based configuration and integrates directly into web pages.

pgloader is a standalone command-line application for migrating data to PostgreSQL databases. Written in Common Lisp, it supports data loading from various sources, including CSV files, and databases like MySQL, SQLite, and MSSQL, providing features such as data reformatting, error handling with rejected data storage, and transaction management for improved data integrity during bulk loads. Key functionalities include schema migration and parallel data loading. The project is intended for system administrators and database engineers tasked with database migration tasks.

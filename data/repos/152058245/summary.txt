`soar` is a command-line tool and standalone application written in Go designed to optimize and rewrite SQL queries for the MySQL database system. Its core functionality includes SQL optimization based on heuristic algorithms, indexing suggestions for complex queries (UPDATE, INSERT, DELETE, SELECT), SQL syntax rewriting, and EXPLAIN information interpretation. The tool supports SQL fingerprinting, compression, and beautification, along with features for merging multiple ALTER requests and customizable SQL rewriting rules. Targeting database administrators and developers, it serves as a developer productivity tool.

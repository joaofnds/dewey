This open-source Python project, named dgtlmoon/changedetection.io, offers a self-hosted web page change detection, restock monitoring, and notification service. It solves the problem of tracking changes in website content, including text, price, and product availability. Key features include visual selector tools for targeting specific parts of webpages, interactive browser steps to perform actions like logging into websites or filling out forms, and restock/price change notifications via various platforms such as Discord, Email, Slack, Telegram, API calls, etc. The technology stack includes Python, Playwright content fetcher, and the project is licensed under Apache License 2.0. It can be used by developers, businesses, or individuals who require real-time monitoring of website changes for various purposes such as price tracking, defacement monitoring, or restock alerts.

**changedetection.io** is a Python-based web scraping and monitoring service that provides automated website change detection and notification capabilities. Built for DevOps, system administrators, and technical users, it monitors web pages for content modifications using HTTP requests and browser automation through Playwright integration. The platform offers CSS/XPath selectors for targeting specific page elements, visual diff comparison tools, and supports interactive browser steps for authenticated sessions and complex workflows. Key features include restock monitoring with price tracking, PDF change detection, and multi-channel notifications via Discord, Slack, Telegram, and email webhooks. The self-hosted solution addresses monitoring needs for e-commerce inventory tracking, security advisory updates, government website changes, and general web content surveillance, providing an alternative to expensive commercial monitoring services.

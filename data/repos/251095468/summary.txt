This repository provides a template for establishing an automated semantic release workflow within a Lerna monorepo, facilitating package publishing to the GitHub Package Registry. It leverages Yarn Workspaces, Conventional Commits, and GitHub Actions to automate versioning, changelog generation, and package publishing upon pushes to the `master` branch. The project, written in JavaScript, enables developers to streamline the release process for multi-package repositories, ensuring consistent versioning and simplified publication within the GitHub ecosystem.  It is a template project aimed at JavaScript developers managing monorepos.

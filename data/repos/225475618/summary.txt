`edamagit` is a Visual Studio Code extension, written in TypeScript, that provides a keyboard-driven Git interface, emulating the functionality of the Emacs-based Magit tool. Its core purpose is to enhance developer productivity by offering a comprehensive and interactive Git workflow directly within VS Code. Key features include a status view for managing changes, file-specific operations, and dispatch commands, with customizable keybindings and support for VSCodeVim integration. This project is a developer productivity tool targeting users of Visual Studio Code who desire a powerful, keyboard-centric Git interaction experience.

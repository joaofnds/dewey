`yabai` is a standalone application, specifically a tiling window manager for macOS, written in C. It provides automated window layout management through a binary space partitioning algorithm, allowing users to control windows, spaces, and displays via a command-line interface and keyboard shortcuts. Key features include focus-follows-mouse, space switching animation control, and extending the number of available spaces. It requires specific macOS system settings and accessibility permissions to function correctly and is aimed at macOS users seeking enhanced window management capabilities.

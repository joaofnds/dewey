Quil is a Clojure library providing a Processing-compatible API for creating visual art, animations, and interactive sketches. It enables users to generate graphics and animations by leveraging the expressiveness of Clojure and the established Processing API, supporting both Clojure and ClojureScript. Key functionalities include `setup`, `draw`, and `sketch` functions to define the artwork's initial state, continuous rendering behavior, and the overall sketch, respectively, with features like support for the standard Processing API, and a wide range of examples. Quil facilitates the creation of generative art, targeting developers and artists seeking a functional, code-based approach to visual creation. It is a library, offering a core API and integrations with Clojure and ClojureScript projects.

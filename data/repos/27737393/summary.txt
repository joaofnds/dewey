`qmk/qmk_firmware` is an open-source keyboard firmware designed for Atmel AVR and Arm USB microcontrollers. Written primarily in C and C++, it provides extensive keyboard support, enabling custom layouts and functionality. This firmware project facilitates the creation of custom keyboard layouts and features, supporting various keyboard models including Clueboard, ErgoDox EZ, and OLKB. As a firmware implementation, it is a low-level software project intended for keyboard enthusiasts and hardware developers.

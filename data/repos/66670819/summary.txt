Crawlee is a TypeScript library designed for web scraping and browser automation. It provides a unified interface for both HTTP and headless browser crawling, enabling developers to build robust and reliable web scrapers. Key functionalities include URL queue management, persistent storage, proxy rotation, headless browser support via Playwright and Puppeteer, and automatic scaling. This library offers features like HTTP/2 support, browser fingerprinting, and a CLI for project bootstrapping, serving as a developer tool for extracting data from the web.

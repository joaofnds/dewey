Taskell is a command-line task manager and Kanban board application written in Haskell, designed for personal project management and developer productivity. It allows users to create and manage task lists, supporting features like vim-style keybindings, Markdown-based storage for easy version control, sub-tasks, and due dates. Key functionalities include importing tasks from Trello boards and GitHub projects, enabling synchronization with external project management tools. As a standalone CLI application, Taskell targets developers and individuals seeking a text-based task management solution for Linux and macOS.

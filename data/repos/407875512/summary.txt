`sttr` is a cross-platform command-line application written in Go designed for string manipulation and transformation. It provides a wide range of functionalities, including encoding/decoding (base64, URL, HTML), hashing (MD5, SHA256), case conversions, line operations, JSON/YAML conversion, markdown-to-HTML conversion, and text extraction. This CLI tool targets developers and power users seeking a versatile and efficient utility for string-related tasks, and it supports installation via various package managers and direct binaries.

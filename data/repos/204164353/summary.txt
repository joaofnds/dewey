Kestra is a Java-based, open-source workflow automation platform designed for orchestrating both scheduled and event-driven workflows. It enables Infrastructure-as-Code practices within data, process, and microservice orchestration through a declarative YAML interface and a rich plugin ecosystem supporting numerous languages and integrations. Key functionalities include a visual UI for building workflows, Git integration for version control, high availability, and support for event-driven triggers, targeting DevOps, data engineers, and developers seeking to automate complex pipelines and tasks. This project is a standalone application providing features akin to Airflow, n8n, and similar platforms.

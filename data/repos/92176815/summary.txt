`jsonista` is a Clojure library providing fast JSON encoding and decoding capabilities. It leverages the Jackson-databind library, utilizing mostly Java code for enhanced performance. The library offers explicit configuration options and supports features like tagged JSON and streaming, making it suitable for web development and data processing tasks. `jsonista` is a library designed to be a faster alternative to existing Clojure JSON libraries and aims to improve developer productivity for Clojure developers working with JSON data.

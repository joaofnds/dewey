Filestash is a web-based file manager application designed for managing data across diverse storage backends. It supports protocols including SFTP, S3, FTP, WebDAV, Git, and others, providing a unified interface for file access and manipulation. Built with JavaScript, it features file sharing, media playback, transcoding, and plugin support for custom functionality like LDAP authentication and QR code generation, built on a core/plugin architecture that enables custom file manager builds. The project type is a standalone web application aimed at users needing a centralized management tool for files stored across multiple cloud providers and network protocols.

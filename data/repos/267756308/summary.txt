`carta` is a Go library designed for mapping SQL query results directly into Go structs, specifically addressing complex data relationships. It excels at handling has-one and has-many relationships without requiring an ORM, supporting common SQL databases like MySQL and PostgreSQL. The library utilizes struct tags for column mapping and manages data type conversions, including support for nested structs and slices for representing complex data structures within the Go application. `Carta` is a developer productivity tool targeting Go developers building applications that interact with SQL databases.

`roughViz` is a JavaScript library designed for creating hand-drawn-style charts in web browsers, focusing on data visualization within the data science and dashboarding domains. It leverages D3v5 and roughjs to generate various chart types, including bar, line, donut, pie, and scatter plots, with customizable stylistic options derived from roughjs, such as roughness, fill styles, and fill weight. The library supports installation via CDN or npm, and includes wrappers for React, Vue, and Python, making it a reusable component intended for frontend developers building data-driven applications and dashboards with a unique visual aesthetic.

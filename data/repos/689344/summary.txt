RetroArch is a cross-platform frontend application for the libretro API, enabling the use of emulators, game engines, and 3D programs as dynamic library "cores". Written in C, it provides a unified interface for audio/video output, input, and application lifecycle management across numerous platforms, including Android, iOS, Windows, Linux, and consoles. Key functionalities include gamepad-centric and touchscreen UI support, multi-pass shader support, real-time rewind, video recording, and run-ahead input latency removal, acting as a framework and standalone application that integrates into various launcher frontends.

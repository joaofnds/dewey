`0x192/universal-android-debloater` is a cross-platform GUI application written in Rust designed to debloat non-rooted Android devices. It utilizes ADB (Android Debug Bridge) to uninstall or disable pre-installed system packages, improving privacy, security, and battery life. Key features include multi-user and multi-device support, export/import functionality for package selections, and detailed action logging. This standalone GUI application targets users seeking to customize their Android experience by removing unwanted system applications.

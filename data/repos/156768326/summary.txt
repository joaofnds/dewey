Watermill is a Go library designed for building event-driven applications, enabling features like event sourcing and CQRS using message streams. It provides a unified interface for publishing and subscribing to messages, supporting various pub/sub implementations, including Kafka, RabbitMQ, NATS, and HTTP. The library offers flexibility through middlewares and plugins, facilitating the creation of resilient and scalable distributed systems. This Go library serves as a framework for developers building event-driven architectures with an emphasis on ease of use, performance, and integration with common messaging systems.

`ddd-forum` is a Hacker News-inspired forum application built with TypeScript, demonstrating Domain-Driven Design (DDD) principles and Clean Architecture. The application features a forum with user authentication, post creation, commenting, and voting functionalities. Utilizing Express.js for the backend, React.js and Redux for the frontend, and Sequelize for database interactions, it showcases a layered architecture separating domain logic, application use cases, and infrastructure concerns. This is a full-stack web application example/template project targeting developers learning or applying DDD and clean architecture with TypeScript.

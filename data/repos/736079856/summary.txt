`precognition.nvim` is a Neovim plugin written in Lua designed to enhance user navigation within text buffers. It leverages virtual text and gutter signs to visualize available movement options, aiding users in discovering and executing motions. The plugin allows users to toggle hints, peek at motions, and disable them for specific filetypes via configuration options. It is a plugin that integrates with the Neovim editor, targeting Neovim users looking to improve text navigation and workflow.

`clojure-ts-mode` is an Emacs major mode providing enhanced Clojure and ClojureScript code editing support, leveraging the Tree-sitter parsing library for improved syntax highlighting, indentation, and navigation. This Emacs Lisp-based plugin offers features including fast and accurate font-locking, indentation strategies (semantic and fixed), structured navigation, and integration with Emacs's `imenu`. It aims to replace the legacy `clojure-mode` by utilizing Tree-sitter to address existing limitations. `clojure-ts-mode` is a developer productivity tool, a plugin/extension for Emacs, targeting Clojure and ClojureScript developers.

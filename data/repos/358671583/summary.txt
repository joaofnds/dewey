`tokyonight.nvim` is a Neovim theme written in Lua, providing a clean, dark visual style for the Neovim text editor. It features multiple theme variations (Moon, Storm, Night, Day) and supports integration with numerous Neovim plugins, including those for LSP, treesitter, and other common extensions. The project type is a Neovim plugin and provides color schemes for terminal emulators such as Alacritty, iTerm, and Kitty, plus a configuration for Fish shell, enhancing the user experience across the development environment. Its target audience is Neovim users seeking a customizable, aesthetic theme for their development workflow.

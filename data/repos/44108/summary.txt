LFE (Lisp Flavoured Erlang) is a front-end for the Erlang compiler that provides a Lisp syntax, enabling developers to write code compatible with standard Erlang. This project is a compiler, REPL, and related tooling for LFE, allowing users to compile, execute, and debug LFE code. Key functionalities include a REPL, Docker support for containerization, and build and installation scripts. It leverages the Erlang ecosystem, targeting developers familiar with Lisp and Erlang, and supporting integration with Erlang/OTP.

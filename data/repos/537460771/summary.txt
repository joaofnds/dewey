`fennel-language-server` provides Language Server Protocol (LSP) support for the Fennel programming language, enabling features like diagnostics, code completion, and go-to-definition within code editors and IDEs. Built in Rust, it functions as a standalone server installable via `cargo`. It integrates with editors such as Neovim via configuration, enabling features for Fennel development. This project is a developer productivity tool, specifically an LSP server, offering features for code analysis and navigation.

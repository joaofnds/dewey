`tokei` is a command-line tool written in Rust for rapidly counting code statistics across numerous programming languages. Its core function is to analyze source code repositories, providing detailed metrics such as file counts, lines of code, comments, and blank lines, grouped by language. Supporting over 150 languages, `tokei` offers flexible output formats including JSON, YAML, and CBOR, and integrates features like file exclusion via `.gitignore` and sorting capabilities. This standalone application provides a developer productivity tool for software project analysis and reporting.

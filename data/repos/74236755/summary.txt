Fathom Lite is an open-source web analytics platform built with Go and Preact that provides privacy-focused website traffic monitoring as an alternative to Google Analytics. Targeting web developers and site administrators in the digital analytics domain, it implements a lightweight tracking solution with a self-hosted dashboard for collecting and visualizing website visitor data. The system uses a Go backend for data processing and storage, a Preact-based frontend for the analytics dashboard, and includes Docker containerization for deployment. It addresses privacy concerns in web analytics by offering cookie-based tracking with minimal data collection, serving developers who need basic website metrics without third-party data sharing or complex enterprise analytics features.

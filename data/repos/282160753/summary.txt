This Clojure repository provides curated lists of names drawn from various domains for use in software development, specifically for naming software components, services, and releases. It facilitates name automation through easily accessible lists stored as files, with recommendations for usage via git snapshots. Contributions are encouraged to expand and refine these name lists, adhering to established quality and naming conventions. This project functions as a developer productivity tool and a collection of curated data intended for augmenting software naming practices.

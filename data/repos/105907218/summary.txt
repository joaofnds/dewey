`lgo` is a Go (golang) project providing an interactive REPL and Jupyter Notebook kernel for data science and interactive Go programming. It allows users to write and execute Go code interactively, leveraging full Go language specifications with code completion and inspection within Jupyter Notebooks. The project integrates with Jupyter Notebooks, supports display of rich media including images and HTML, and uses the Go compiler (gc). It is a developer productivity tool and interactive environment for Go, targeting Go programmers and data scientists interested in exploratory data analysis and interactive coding workflows.

`WMR` is a JavaScript build tool designed for rapidly developing modern web applications. It provides an all-in-one development environment featuring support for ES modules, hot reloading, TypeScript, CSS Modules, and static file serving. Built around Rollup for optimized production output, it enables static HTML prerendering and offers a built-in development server. This project acts as a developer productivity tool, streamlining the front-end development workflow, primarily for developers using Preact and targeting modern web browsers.

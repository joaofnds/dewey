`Torn` is a split keyboard design utilizing through-hole components, based on QMK firmware. Functionality centers around providing a customizable, ergonomic keyboard layout and firmware. Key features include compatibility with VIA for keyboard configuration, and integration with an Atmega328P CPU. The project is a hardware design, firmware, and associated documentation set, intended for hobbyists and mechanical keyboard enthusiasts interested in building their own split keyboards. The project utilizes QMK firmware and the VUSB library.

n8n is a TypeScript-based, fair-code workflow automation platform enabling users to visually build and deploy automated workflows across 400+ integrations, with built-in AI capabilities. This platform allows for self-hosting or cloud deployment and offers features such as custom code execution (JavaScript/Python), support for npm packages, and integration with LangChain for AI agent workflows. It provides an extensible, enterprise-ready architecture, including advanced permissions, SSO, and air-gapped deployment options, aimed at technical teams seeking a low-code solution for automating tasks and connecting APIs. This is a standalone, web-based application and CLI tool.

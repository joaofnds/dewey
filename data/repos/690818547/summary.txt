**Nue** is a JavaScript-based web framework focusing on adherence to web standards, actively developed since 2023. Its primary goal is to address the increasing complexity in modern web development by streamlining the ecosystem from scratch, aiming for a radically simpler stack. Key features include HTML, CSS, JS, and WASM support, with an emphasis on design system and markdown-flavored documentation. The project is currently a framework, targeting developers seeking a standards-first approach to web development. Notable aspects include its active development, focus on exposing hidden complexities, and ambitious vision to rebuild the entire ecosystem by 2025.

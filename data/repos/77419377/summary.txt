`ctop` is a command-line application providing a real-time, top-like interface for monitoring container metrics. Built in Go, it offers a condensed overview of Docker and runC container performance, displaying metrics in a grid format and enabling single container views. Users can filter, sort, and configure displayed data, utilizing keybindings for interaction, with support for saving configurations. This standalone CLI application targets system administrators and developers working with containerized applications, providing an interactive tool for container resource monitoring.

`learndatalogtoday` is an interactive tutorial application for learning Datalog. This standalone application, built with Clojure, enables users to explore and understand Datalog through an interactive learning experience. The project utilizes Leiningen for dependency management and build processes, producing a self-contained `.jar` executable for local execution. The application runs as a web server, accessible through a browser, likely providing a user interface for interacting with and learning Datalog concepts.

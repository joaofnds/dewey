This repository provides proof-of-concept implementations for Go web applications, demonstrating best practices for building maintainable and testable systems. It showcases a hexagonal architecture, utilizing dependency injection with Uber's FX framework to manage dependencies and promote loose coupling. Key functionalities include GRPC server implementations, HTTP handling with Fiber, Kafka integration for message queuing, and integration tests. It is a Go-based project, likely an example or template, aimed at Go developers learning or exploring modern web application architecture and related technologies such as gRPC and Kafka.

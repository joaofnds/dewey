Cryogen is a static site generator written in Clojure, designed to create websites and blogs. It supports Markdown and AsciiDoc content, with features including tag management, table of contents generation, Sass/SCSS compilation, and RSS feed generation. The project utilizes Clojure, incorporates Selmer templating, and leverages libraries like Highlight.js for syntax highlighting.  Users can deploy the generated HTML output to static web servers such as Nginx. This is a command-line tool/template project aimed at developers who need to create static websites.

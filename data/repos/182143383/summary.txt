`sshcode` is a Go-based CLI tool designed to automatically install and run code-server on a remote server over SSH, effectively enabling remote VS Code development. It synchronizes local VS Code extensions and settings to the remote server, streamlining the development workflow. The tool supports Linux, macOS, and WSL, and targets Linux x86_64 servers with glibc, facilitating a seamless, native-like VS Code experience. This standalone CLI application simplifies remote development by offering a streamlined method to leverage VS Code on remote machines.

`Renovamen/playground-macos` is a web-based portfolio application simulating the macOS graphical user interface. Developed with React and TypeScript, it leverages UnoCSS for styling and Zustand for state management. The application showcases a macOS-like environment, complete with interactive elements and simulated features like FaceTime, Typora integration (built with Milkdown), and a battery status display. This is a standalone web application, targeting frontend developers and showcasing a personal portfolio through an interactive and visually engaging user interface.

This repository provides example queries and rules for interacting with a Datomic database populated with a subset of the MusicBrainz dataset. It showcases Datomic's capabilities for querying and managing time-based facts, utilizing Clojure for its code base. The project includes instructions, sample queries, and a schema definition to facilitate learning and experimentation with Datomic and the MusicBrainz data, making it an example project for Datomic users interested in exploring data modeling and querying in a music metadata domain. It is a sample project intended for Datomic users and those interested in exploring its data modeling and querying capabilities with an example dataset.

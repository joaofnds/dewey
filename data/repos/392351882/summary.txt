`cmp-path` is a Lua plugin providing path completion functionality for the Neovim text editor's `nvim-cmp` completion framework. It allows users to insert filesystem paths during text editing. Key features include options for trailing slashes in completed directory names and in completion menu labels, along with a configurable base directory via a `get_cwd` function. This project is a plugin/extension designed to enhance the developer experience within the Neovim editor.

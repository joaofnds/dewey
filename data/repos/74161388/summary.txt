Tilex is an open-source "Today I Learned" (TIL) web application, built with Elixir and Phoenix, designed for the collaborative sharing and accumulation of knowledge. Key functionalities include a 200-word limit for posts and authentication via Ueberauth and Google for user management. It is deployable to Heroku and utilizes PostgreSQL for data storage. The project is a standalone web application and is targeted towards software developers and engineers interested in learning about Elixir, Phoenix, and web application development.

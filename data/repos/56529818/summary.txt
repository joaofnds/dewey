Martian is a Clojure and ClojureScript library providing an HTTP abstraction layer, streamlining interactions with APIs described by OpenAPI/Swagger specifications or custom data definitions. It enables developers to interact with HTTP endpoints by simply calling operations with parameters, abstracting away HTTP details. Key features include support for various HTTP client libraries, interceptor-based extensibility, request and response validation, and integration with re-frame. This library is designed to simplify building HTTP clients and enhance developer productivity. It offers testing, recording, and playback capabilities via companion libraries.

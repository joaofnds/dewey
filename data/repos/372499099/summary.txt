**Kino** is an Elixir library for Livebook, providing client-driven interactive widgets to render rich and dynamic output directly from Elixir code. Its core capabilities include data charting using Vega-Lite, making it suitable for machine learning and data analysis domains. Key features include automatic retries, request/response interceptors, and built-in timeout management, ensuring robust and responsive user interactions. This library is designed to be easily integrated with other Livebook packages such as Bumblebee, Explorer, Maplibre, Slack, and Benchee for specific use cases. The project type is a library, and it targets developers working on Elixir projects who require interactive visualizations in their applications.

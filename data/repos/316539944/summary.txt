`Proxify` is a standalone, Go-based HTTP/HTTPS and SOCKS5 proxy tool designed for capturing, manipulating, and replaying network traffic. It provides features including TLS MITM support, request/response filtering and manipulation using a DSL, upstream proxy support, and traffic dumping to various output formats including JSONL and YAML.  Key functionalities include a built-in DNS server for hostname mapping and traffic replay capabilities integrated with tools like BurpSuite. The project is geared towards security professionals and developers involved in web application security, penetration testing, and network traffic analysis.

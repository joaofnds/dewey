This repository houses a collection of Backstage plugins developed by Roadie, a SaaS Backstage solution provider. These plugins, written in TypeScript, extend the functionality of Backstage, an open platform for creating developer portals. Key features include integrations for various tools like AI assistance, GitHub pull requests, AWS Lambda, Jira, and security insights, enabling enhanced developer workflows and platform management. It functions as a plugin/extension project, designed for developers utilizing or extending Backstage developer portals.

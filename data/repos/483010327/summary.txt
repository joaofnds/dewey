`mist` is a Gleam-based HTTP server library designed for building web applications. It provides functionalities for handling HTTP requests, including routing, serving static files, supporting WebSockets, and processing form data, with a focus on streaming request bodies. The library exposes `mist.start_http` and `mist.start_https` functions built upon a `Builder` pattern for configuration, and offers features like chunked responses and content type handling. This library is geared towards Gleam developers needing a performant and flexible web server for Gleam-based web projects.

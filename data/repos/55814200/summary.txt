`archiver` is a Go library (DEPRECATED) focused on archive creation, extraction, and compression/decompression of various formats. Supporting diverse archive formats like 7zip, tar, zip, and several compression algorithms including gzip, bzip2, and zstandard, it enables developers to manipulate archived data within Go applications. It provides streaming capabilities and is designed for developer productivity, with functionality intended for streamlining file archival and retrieval tasks. Its successor is `mholt/archives`, reflecting API and performance improvements.

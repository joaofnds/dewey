`pdfshot` is a standalone application designed to generate PDF screenshots of web pages. Built using Clojure and ClojureScript, it leverages Docker for containerization and Puppeteer for headless browser automation. Functionality includes capturing screenshots from specified URLs, supporting optional waiting for specific page elements to render before capture, and exposing a simple API via HTTP. This project provides a CLI and a Docker image, targeting web developers needing a tool for automated PDF generation from web content.

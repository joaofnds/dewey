CodeEdit is an open-source, macOS-native code editor built with Swift and SwiftUI, providing a feature-rich alternative to existing solutions. It delivers functionalities including syntax highlighting, code completion, project management, debugging tools, Git integration, and extension support. Designed as a standalone application, it aims to offer a lightweight and performant experience tailored for macOS developers, drawing inspiration from and seeking to emulate Apple's interface design principles, with a focus on community contributions and open-source development. This project is composed of multiple repositories with specialized responsibilities.

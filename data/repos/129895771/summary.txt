Kaocha is a full-featured, next-generation test runner for Clojure and ClojureScript projects, designed for developer productivity. It facilitates testing with features like filtering by test names and metadata, a watch mode for automatic test re-runs, customizable reporting, and test order randomization. The project supports plugins for extending functionality and integrates with various testing frameworks, including clojure.test. Kaocha is a developer tool, providing a CLI and library for comprehensive test execution within Clojure projects.

`nnn` is a full-featured, orthodox terminal file manager written in C, designed to enhance developer productivity in CLI environments. It provides core functionalities including file navigation, disk usage analysis, file preview, and batch renaming, and extends its capabilities through a plugin architecture. Key features include efficient resource utilization (low memory footprint and fast execution), support for a variety of file system operations, and integration with popular tools like `vim`. This standalone, multi-platform application is targeted towards users seeking a lightweight and customizable file management solution within the terminal, including support for environments like Android (Termux), Linux, macOS, and WSL.

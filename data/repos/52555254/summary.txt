`goose` is a database migration tool written in Go, providing both a command-line interface (CLI) and a library for managing database schema changes. It facilitates database migrations using SQL scripts and Go functions, supporting multiple database systems including PostgreSQL, MySQL, SQLite, and others. Key features include out-of-order migrations, environment variable substitution in SQL, and embedded migration support, allowing developers to version and deploy schema changes effectively. This project functions as a standalone CLI application and Go library, targeting developers managing and deploying database schema updates.

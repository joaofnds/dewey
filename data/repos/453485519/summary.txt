PgCat is a high-performance PostgreSQL connection pooler and proxy, written in Rust, designed for improved database scalability and resilience. It provides features including transaction and session pooling, multi-threaded operation via Tokio, and load balancing of read queries across replicas. Core functionalities include automatic failover, sharding capabilities using extended SQL syntax, and mirroring for testing. PgCat offers both Prometheus and administrative database statistics reporting, and supports SSL/TLS and client/server authentication. This is a standalone application targeting database administrators and developers, enabling efficient database connection management and advanced features such as sharding and failover for PostgreSQL deployments.

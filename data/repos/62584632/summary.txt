`Coherence` is an Elixir library providing a full-featured and configurable authentication system for Phoenix web applications. It offers modules for database-backed authentication, registration, confirmation, password recovery, and session management, including features like invitation-based user onboarding, login statistics tracking, account locking, and persistent "remember me" functionality.  The library generates customizable boilerplate code, including database migrations and Phoenix-specific controller, view, and template files, enhancing developer productivity. This is a library and framework extension, designed for Elixir/Phoenix developers to streamline authentication implementation in their web applications.

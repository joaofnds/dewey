This repository provides a detailed, step-by-step tutorial for manually bootstrapping a Kubernetes cluster, focusing on the underlying components and their interactions. It guides users through the process of setting up essential Kubernetes components like etcd, control plane elements, and worker nodes, using a "hard way" approach with no automation scripts. The project's core function is educational, aimed at individuals seeking a deep understanding of Kubernetes fundamentals and cluster configuration. It involves manual setup across multiple virtual machines, covering certificate authority provisioning, configuration file generation, and pod network routing, targeting system administrators and those learning Kubernetes internals.

This repository provides dotfiles, configurations for a macOS environment, managed with Git and GNU Stow. It encompasses configurations for Neovim, Tmux, Kitty, Bash, and yabai, including a Homebrew configuration for package management. Key functionalities include customizing the user's terminal, text editor, and window management, enhancing developer productivity on macOS. Built primarily using Lua (for Neovim config) and Bash scripting, alongside configurations written in formats specific to each tool. This project is a set of configuration files, serving as a template or example for customizing a developer's macOS setup.

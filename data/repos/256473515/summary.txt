`protesilaos/modus-themes` is an Emacs Lisp project providing highly accessible themes for the GNU Emacs text editor, adhering to WCAG AAA color contrast standards. The repository offers customizable themes, optimized for users with color vision deficiencies, that provide broad support for a wide range of Emacs packages. This library-type project serves as an extension to Emacs, enabling users to improve the visual accessibility and customizability of their editing environment.

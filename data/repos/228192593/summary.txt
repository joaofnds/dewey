`hakrawler` is a fast, Go-based command-line web crawler designed for rapid discovery of URLs and JavaScript file locations within web applications, targeting the cybersecurity domain. It facilitates endpoint and asset reconnaissance, supporting functionalities like subdomain inclusion, proxy configuration, and output formatting options (JSON, unique URLs). The tool utilizes the Go programming language and leverages the `Gocolly` library for crawling. It is a standalone CLI application primarily intended for security professionals engaged in penetration testing, bug bounty hunting, and open-source intelligence (OSINT) gathering.

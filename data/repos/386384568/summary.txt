`ots` is a Go-based command-line interface (CLI) tool designed for secure secret sharing using end-to-end encryption and one-time URLs. It allows users to share sensitive information like API keys and passwords, which are destructed after viewing or a specified expiry period. The application provides features for managing secret residency via region selection and integrates with standard input pipelines. This standalone application targets developers and DevOps teams seeking a secure and ephemeral secret management solution.

`conjure` is a Neovim plugin providing interactive evaluation capabilities for multiple programming languages, including Clojure, Fennel, Janet, Racket, Hy, and Scheme dialects. Its core functionality allows users to interactively execute and experiment with code directly within a Neovim environment, facilitating a REPL-like workflow. The plugin is primarily written in Fennel, leveraging the Neovim API to integrate evaluation capabilities. It targets developers utilizing Neovim for development in the supported languages, enhancing their interactive coding and debugging experience.

Cap is an open-source, cross-platform desktop application providing screen recording, editing, and sharing functionalities, positioned as an alternative to Loom. Built as a monorepo using TypeScript, it leverages Tauri for the desktop application, a Next.js web application for the frontend, and SolidStart for the desktop UI. Key technologies include React for shared UI components, Drizzle ORM for database interaction, and Rust for the Tauri backend. This project aims to offer a developer-friendly solution for creating and sharing video messages.

drawDB is a web-based application providing a free and intuitive database schema editor and SQL generator. Built with JavaScript, React, and utilizing Tailwind CSS for styling, it allows users to visually design database entity relationship diagrams (ERDs) and generate SQL scripts for various database systems including PostgreSQL, MySQL/MariaDB, SQL Server, Oracle, and SQLite. Key features include diagram creation, SQL script export, and a user-friendly interface. This project serves as a standalone web application, enabling database administrators and developers to design and document database schemas efficiently without requiring account creation.

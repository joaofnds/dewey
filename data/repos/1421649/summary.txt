`doric` is a Clojure library designed for rendering tabular data in various formats. It facilitates the generation of pretty, emacs org-mode-style tables, as well as raw, CSV, and HTML tables, from nested Clojure data structures. Key features include customizable column formatting, alignment control, and the ability to define custom table formats via user-provided functions.  It functions as a library providing a function named `table` and caters to Clojure developers who need to display structured data.

Casdoor is an open-source, UI-first Identity and Access Management (IAM) / Single-Sign-On (SSO) platform written in Go. It provides a web-based interface and supports various authentication protocols including OAuth 2.0, OIDC, SAML, CAS, LDAP, SCIM, WebAuthn, TOTP, MFA, and RADIUS. Functionality includes user authentication, authorization, and identity management, catering to diverse integration needs. This is a standalone web application project targeting developers and system administrators seeking centralized authentication and access control solutions.

Casdoor is an open-source Identity and Access Management (IAM) and Single Sign-On (SSO) platform built in Go, targeting enterprise web applications and distributed systems requiring centralized authentication. It implements comprehensive authentication protocols including OAuth 2.0, OIDC, SAML, CAS, LDAP, SCIM, and RADIUS, while supporting modern security features like WebAuthn, TOTP, MFA, and biometric authentication. The platform provides a web-based administrative interface for managing users, organizations, and access policies, integrating with enterprise directory services like Active Directory and Google Workspace. Designed for DevOps teams and system administrators, Casdoor solves the complex problem of unified identity management across multiple applications and services, offering deployment flexibility through Docker and Kubernetes while providing REST APIs for custom integrations and automated user provisioning workflows.

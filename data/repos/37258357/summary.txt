Hunspell is a C++ library and command-line tool designed for spell checking and morphological analysis, primarily targeting languages with complex morphology and compounding. It provides features such as Unicode support, customizable suggestion algorithms, handling of complex compounds, stemming, and morphological analysis/generation. The project includes both a C++/C API and a terminal-based user interface using the Curses library, alongside various parsing options. It is a library, used by applications like LibreOffice, Mozilla Firefox, and Google Chrome; the command-line tool offers functionality for dictionary development, analysis, and testing, and depends on libraries like libiconv, gettext, and ncurses/readline.

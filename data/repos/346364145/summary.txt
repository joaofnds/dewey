Based.cooking is a static recipe website built with Hugo static site generator and a custom CSS theme called Lugo, targeting the culinary/food content domain. The repository serves as a community-driven recipe collection platform where contributors submit markdown-formatted recipes following strict formatting guidelines, with images stored as optimized WebP files under 100KB. The site emphasizes traditional cooking content, implements no tracking or advertising, and operates under public domain licensing. Primary users are cooking enthusiasts and web developers contributing recipe content through GitHub pull requests, with the technical stack focusing on static site generation, markdown content management, and performance-optimized image handling for food photography presentation.

FishUI is a C++ GUI library for CutefishOS, built upon Qt Quick Controls 2. Its core functionalities include light and dark mode support, borderless window management (XCB-based), window blurring and shadow effects, and a desktop-level menu implementation. The library provides styling for Qt Quick controls and integrates with the CutefishOS environment, offering a set of GUI components and features. It is intended for application developers building user interfaces on CutefishOS.

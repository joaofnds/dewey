This repository serves as a digital asset collection containing Go programming language mascot artwork created by Ashley McNamara, based on Renee French's original Gopher design. Targeting the Go developer community and programming conference organizers, it provides print-ready PNG files for creating stickers, promotional materials, and community merchandise. The artwork collection addresses the need for standardized, high-quality Go branding assets within the software development ecosystem, particularly for meetups, conferences, and open-source project branding. Licensed under Creative Commons Attribution-NonCommercial-ShareAlike 4.0, it enables Go developers, community managers, and tech event organizers to access consistent visual branding materials while maintaining legal compliance for non-commercial community building activities in the programming language domain.

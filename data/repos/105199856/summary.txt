`Reticulum` is an Elixir-based web server designed for the `Hubs` social mixed reality platform, functioning as a hybrid game networking and web API server. Key functionalities include handling web requests and API interactions for the `Hubs` application and providing the necessary infrastructure to support the platform's features. It relies on Elixir, Phoenix framework, and PostgreSQL for data persistence, with dependencies including Erlang, Ansible, and libraries for game networking. This project is a standalone web application, specifically a backend server, intended for developers working on the `Hubs` platform and related services.

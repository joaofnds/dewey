`org-kanban` is an Emacs Lisp extension designed to implement Kanban boards directly within org-mode files. It provides functionality for managing tasks and workflows using a visual Kanban table representation within the org-mode ecosystem. The project is implemented in Emacs Lisp, leveraging org-mode's data structures and editing capabilities. This is an Emacs plugin/extension intended for users of org-mode, offering an integrated task management solution.

`FiloSottile/age` is a Go-based command-line utility and library for secure file encryption, designed with a focus on simplicity and modern cryptographic practices. It provides functionalities for encrypting and decrypting files using explicit keys, supporting multiple recipients, passphrases, and SSH keys. Featuring UNIX-style composability, `age` allows for integration into shell pipelines and offers interoperability with Rust and TypeScript implementations. This standalone application and library aims to offer a secure and user-friendly alternative for file encryption, targeting developers and system administrators needing a robust and versatile encryption solution.

`Tidy Viewer (tv)` is a cross-platform, Rust-based command-line interface (CLI) tool designed for pretty-printing and visualizing tabular data from CSV files within a terminal environment, with an emphasis on maximizing readability. It provides features like customizable column styling, significant digit printing to reduce decimal clutter, NA value handling, column overflow logic, and long string truncation for improved data presentation. This standalone application offers a user-friendly alternative to standard `column` and related tools, targeting data scientists, analysts, and anyone working with CSV data, enabling better data exploration and comprehension directly from the command line.

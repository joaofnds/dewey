Caddy is a fast and extensible, multi-platform HTTP/1-2-3 web server implemented in Go, designed for automatic HTTPS and privacy-focused web serving. Its core functionality includes automatic certificate management via ACME, reverse proxy capabilities, and support for HTTP/3. Key features are easy and dynamic configuration through the Caddyfile and JSON API, modular architecture for extensibility via plugins, and native support for HTTP/1.1, HTTP/2, and HTTP/3. This standalone application targets system administrators and developers needing a secure, high-performance web server.

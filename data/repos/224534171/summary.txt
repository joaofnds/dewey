Phoenix LiveDashboard is a real-time performance monitoring and debugging tool for Phoenix developers, providing insights into application metrics, request logging, and system resource utilization. Built on Elixir and LiveView, it offers a web-based dashboard to visualize telemetry events, inspect OS data, manage applications, processes, and storage (Ecto). This project type is a standalone web application that integrates directly with Phoenix applications through configuration within the router and supports distributed Erlang for cross-node monitoring. The target audience is Phoenix developers seeking to improve application observability and debug issues.

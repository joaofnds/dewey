Xournal++ is a cross-platform, standalone GUI application for handwriting notetaking and PDF annotation, written in C++ using GTK3. It allows users to create handwritten notes, annotate PDFs, and utilize pen input from devices like Wacom tablets. Key functionalities include paper backgrounds, PDF selection and linking, exporting to various formats (SVG, PNG, PDF), diverse drawing tools, LaTeX support, page management features, audio recording, and plugin support via Lua scripting. The project is a desktop application, targeting users who need to take handwritten notes and annotate documents on various operating systems, including Linux, macOS, and Windows.

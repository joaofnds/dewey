Xournal++ is a cross-platform handwriting and note-taking application built in C++ with GTK3, targeting digital pen and stylus users across Linux, macOS, and Windows. The software serves the digital documentation and academic domains by providing PDF annotation capabilities, pressure-sensitive input support for Wacom and similar tablets, and vector-based drawing tools. Core technical features include SVG/PNG/PDF export, LaTeX integration, Lua-based plugin system, and real-time stroke recognition. The application addresses the need for professional digital note-taking and document annotation workflows, particularly for students, researchers, and professionals who require precise handwritten input with PDF markup capabilities. Built with modern C++ and leveraging Cairo graphics rendering, it provides a native desktop alternative to web-based annotation tools.

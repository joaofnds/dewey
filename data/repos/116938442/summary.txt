Yaegi is a Go interpreter that enables the execution of Go scripts and plugins within embedded environments or interactive shells, built entirely in Go using the standard library. It provides a complete implementation of the Go specification, offering features like a simple interpreter API, access to Go and runtime resources, and security measures by default. This project functions as a library and a command-line executable, supporting dynamic extension frameworks and interactive REPL sessions. It caters to developers seeking to embed Go scripting capabilities or create dynamic plugin systems.

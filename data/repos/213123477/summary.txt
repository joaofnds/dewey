`go-rel/rel` is a Go-based ORM (Object-Relational Mapper) designed for modern database access, particularly within layered architectures. It provides a testable database layer with a built-in testing package, supporting features such as nested transactions, an extendable query builder, eager loading, composite primary keys, and multi-adapter support. This library offers schema migration capabilities and aims to streamline database interactions for Go developers. It functions as a database interaction library, facilitating the development of backend applications that interact with SQL databases.

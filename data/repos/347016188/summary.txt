`switches.mx` is an open-source project representing the website and database for MX mechanical keyboard switches, built with Statamic, a Laravel PHP CMS. The repository contains the website's codebase and content, stored in YAML and Markdown files within a `/content` directory, enabling users to browse and access information about keyboard switches. The project is a standalone, web-based application, requiring PHP and leveraging Laravel Homestead for local development, including dependencies managed via Composer. The primary functionality is centered around displaying and organizing data, with setup instructions targeting developers familiar with Laravel and related technologies.

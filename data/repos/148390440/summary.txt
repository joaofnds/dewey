`kaf` is a command-line interface (CLI) tool written in Go for interacting with Apache Kafka clusters. It provides functionalities for managing and inspecting Kafka resources, including cluster configuration, node discovery, topic management (listing, describing), partition and replica inspection, and consumer group operations. Key features include offset resets, message production from standard input, and shell autocompletion for various shells. Primarily a developer tool, it aims to improve productivity when working with Kafka.

Gogs is a self-hosted Git service written in Go, designed for painless setup and management. It provides a web interface for user, organization, and repository management, alongside features like SSH/HTTP/HTTPS access, webhooks, Git hooks, issues, pull requests, and wikis. The project supports multiple database backends (PostgreSQL, MySQL, SQLite3) and includes features such as a web editor, Jupyter Notebook rendering, and authentication integrations (SMTP, LDAP, GitHub.com). This standalone application facilitates self-hosting Git repositories, targeting developers and system administrators seeking a private or internal code hosting solution.

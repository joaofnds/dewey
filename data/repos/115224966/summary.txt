`cep21/circuit` is a Go library providing an efficient and feature-complete implementation of the circuit breaker pattern, designed to enhance the resilience of Go applications against failures in dependent services. It offers comprehensive metric tracking, integration with `context.Context`, and supports runtime configuration changes, making it suitable for production environments. Key features include customizable state transition logic, SLO tracking, and support for Hystrix dashboards and expvar, enabling monitoring and management of circuit health. This library targets Go developers building robust and fault-tolerant backend systems.

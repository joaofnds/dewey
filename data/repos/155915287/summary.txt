Effection is a TypeScript library providing structured concurrency and effect management for JavaScript applications, enabling robust resource handling and cancellation within concurrent code. It facilitates asynchronous programming by leveraging algebraic effects and async/await, targeting Node.js, browser environments, and Deno. This library offers a task-runner implementation designed to simplify building concurrent applications and is geared towards JavaScript developers seeking improved concurrency control and resource management in their projects. It is packaged and distributed as an npm module.

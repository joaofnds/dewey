Rocket.Chat is an open-source, self-hostable team communication platform built with TypeScript and Meteor.js that provides real-time messaging, video conferencing via WebRTC, and omnichannel customer service capabilities. Targeting enterprises and organizations requiring data sovereignty, it serves as a Slack alternative with enhanced privacy controls and customization options. The platform features a comprehensive REST API, extensible Apps Engine framework for custom integrations, mobile clients built with React Native, and supports both cloud and on-premises deployment models. Primary users include system administrators deploying internal communication infrastructure, developers building custom chat integrations, and organizations in regulated industries requiring complete control over their communication data and compliance requirements.

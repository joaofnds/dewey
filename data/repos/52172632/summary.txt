`evil-multiedit` is an Emacs Lisp package providing multiple cursor support for Emacs's `evil-mode`, integrating `iedit-mode` functionality. It enables interactive selection and simultaneous editing of multiple text matches within a buffer. Core features include matching selections, words, and symbols, along with region management, keybindings for navigation and modification, and an ex command interface. This Emacs plugin targets users of `evil-mode` and Emacs, enhancing their text editing capabilities with multi-cursor operations, and is distributed via MELPA.

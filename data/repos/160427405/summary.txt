Plausible Analytics is a privacy-focused web analytics platform built with Elixir/Phoenix that serves as a GDPR-compliant alternative to Google Analytics. The application uses ClickHouse for time-series data storage, PostgreSQL for relational data, and TailwindCSS for the frontend interface. It provides website traffic measurement, conversion tracking, and user behavior analytics through a lightweight JavaScript tracking script and events API. The system targets web developers, marketing professionals, and website owners who need privacy-compliant analytics without cookies or personal data collection. Key features include real-time dashboards, goal tracking, team collaboration, and Google Search Console integration, specifically addressing the need for privacy-first web analytics in the post-GDPR digital landscape.

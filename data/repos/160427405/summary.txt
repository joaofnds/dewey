Plausible Analytics is an open-source, privacy-focused web analytics application providing a lightweight alternative to Google Analytics. Built with Elixir and Phoenix, it offers core functionalities including website traffic tracking, goal and conversion tracking, email/Slack reporting, and support for Single Page Applications (SPAs). Its architecture utilizes a PostgreSQL database for general data and ClickHouse for statistical data, employing TailwindCSS for styling and React for the interactive dashboard. This is a standalone web application, offering both a managed cloud service and a self-hosted community edition, targeted at website owners and marketing professionals seeking privacy-respecting web analytics.

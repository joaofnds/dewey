Frab is a web-based conference management system, primarily designed for organizing and managing conference submissions, talks, speakers, and schedules. Built as a Ruby on Rails application, it features functionalities for collecting submissions, managing speaker and talk details, and creating event schedules, with support for localization in multiple languages and integration with ticket servers like OTRS and Redmine. The system offers rake tasks for conference exports, static schedule generation, and email management, with deployment options including Docker and PaaS platforms. It is a standalone web application intended for conference organizers.

Netcap is a Go-based framework designed for secure and scalable network traffic analysis, focusing on intrusion detection and security monitoring. It converts network packets into platform-neutral, type-safe audit records, suitable for storage, network exchange, and machine learning applications. Key functionalities include packet capture, protocol parsing, and data export, offering 58 audit record types and integrations for Prometheus metrics and Maltego for enhanced analysis. The framework comprises multiple CLI tools, including a capture agent, collection server, and data transformation utilities, targeting researchers and security professionals involved in network analysis and anomaly detection.

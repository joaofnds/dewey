`jnv` is a standalone, interactive command-line application written in Rust, designed for navigating and filtering JSON data. It functions as an interactive JSON viewer and `jq` filter editor, enabling users to apply `jq` filters and view formatted JSON with syntax highlighting. Key features include auto-completion for `jq` filters, configurable settings via TOML files for UI customization and keybinds, and the ability to handle JSON from files or standard input, including support for JSON Lines. It targets developers and system administrators working with JSON data, providing a CLI-based tool for inspecting and manipulating JSON structures.

TiDB is an open-source, cloud-native, distributed SQL database designed for modern applications, offering high availability, horizontal scalability, and strong consistency. Built in Go, it provides features including distributed transactions, hybrid transactional/analytical processing (HTAP) with row-based (TiKV) and columnar (TiFlash) storage engines, and MySQL compatibility. This standalone database project aims to provide a scalable and reliable solution for managing data in cloud environments, targeting developers and system administrators needing a robust and compatible database system.

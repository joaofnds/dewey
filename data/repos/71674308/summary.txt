This repository provides theming support for the i3 tiling window manager using base16 color schemes. It offers Mustache-based configuration templates and facilitates the integration of base16 color palettes into i3 configurations, enabling users to customize their desktop appearance. Functionality includes fetching pre-configured theme files or color variable sets using `curl` to generate or update the i3 config. This project is a set of configuration templates and utilities designed to enhance the user experience of i3, targeting users who wish to personalize their i3 environment with base16 themes.

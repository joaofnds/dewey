`elasticsearch-rails` is a Ruby library providing integrations for Elasticsearch within ActiveModel/Record and Ruby on Rails applications. It offers model-level search integration, a repository pattern-based persistence layer, and wrappers for search results and ActiveRecord relations. The library facilitates features like indexing, searching, and data import, and includes integration with Rails' instrumentation. This project is a collection of Ruby gems that provide an integration layer between Elasticsearch and Rails, primarily targeting Ruby on Rails developers.

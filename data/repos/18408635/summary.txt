This repository provides the original Assembly source code for the Apollo 11 Guidance Computer (AGC) utilized in the Command Module (Comanche055) and Lunar Module (Luminary099) for the Apollo 11 mission. It serves as an archive of this historical software, enabling access to the core logic that powered the Apollo 11 mission's guidance and navigation systems. The code is written in Assembly language and is intended for archival and educational purposes, facilitating the study and potential compilation of the original AGC programs. The project type is a source code archive focused on the AGC's software, a core component of the Apollo 11 space mission.

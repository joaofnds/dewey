`clj-github-app` is a Clojure library designed to facilitate the implementation of GitHub Apps. It provides functionalities for validating webhook payloads using signature verification and an API client with an HTTP connection pool for interacting with the GitHub API. Key features include secure signature checking for webhooks, an access token manager, and client methods for authenticating as a GitHub App or an installation. This library simplifies tasks related to GitHub App development, targeting Clojure developers building applications that integrate with GitHub's ecosystem, and offering a foundation for handling authentication and webhook events within Clojure projects.

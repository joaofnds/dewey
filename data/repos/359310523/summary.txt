`navigator.lua` is a Neovim plugin designed for code analysis and navigation, enhancing developer productivity. Written in Lua, it leverages Language Server Protocol (LSP) and Treesitter to provide features including diagnostics viewing, symbol navigation, reference searching, and code context analysis. Key functionalities encompass out-of-the-box LSP setup, Treesitter-based symbol search for large files, and a UI using floating windows for managing and navigating symbols. This plugin targets Neovim users seeking an integrated development environment (IDE)-like experience, enhancing code exploration and debugging capabilities within the editor.

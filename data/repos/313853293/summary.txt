CloudQuery is a cloud governance platform designed to discover, monitor, secure, and optimize cloud infrastructure by extracting and integrating data from various sources using SQL. Built with Go and leveraging concurrency, it functions as an ELT (Extract, Load, Transform) platform, enabling high-performance data ingestion and sync capabilities to various data destinations. Key features include Cloud Security Posture Management (CSPM), cloud asset inventory creation, and attack surface management, targeting cloud engineers and security professionals. This project is a standalone, open-source CLI and framework with an accompanying SDK, offering a wide range of integrations for major cloud providers and data sources.

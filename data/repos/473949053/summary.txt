FrankenPHP is a modern application server for PHP, built with Go and leveraging the Caddy web server. It enables high-performance PHP applications by providing features such as Early Hints, worker mode, and real-time capabilities, along with native support for HTTP/2 and HTTP/3. This project functions as a standalone binary, Docker image, and Homebrew package, allowing users to serve PHP applications, including popular frameworks like Laravel and Symfony, efficiently. Furthermore, it can be utilized as a Go library to embed PHP functionality within Go applications using `net/http`.

`org-web` is a web application implementation of the Org-mode plain text markup system, designed for web and mobile accessibility. Built with React and Redux, it provides a web-based interface for creating, editing, and managing Org-mode files. The application synchronizes data with cloud storage providers like Dropbox and Google Drive, enabling user data portability. This JavaScript-based project allows users to interact with their Org-mode files within a web browser, effectively offering a web-based Org-mode editor.

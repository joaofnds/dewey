Natalie is a work-in-progress Ruby compiler utilizing C++ and gcc/clang for backend compilation. Functionality includes an ahead-of-time compiler and a REPL with incremental compilation capabilities. The project allows users to run Ruby scripts, compile files to executables, and execute tests, targeting Ruby developers seeking a performant Ruby implementation. The architecture relies on Ruby and C++ with dependencies including autoconf, automake, libtool, GNU make, and Ruby 3.2+ with dev headers. It's a standalone application with CLI functionalities like a REPL, script execution, and compilation.

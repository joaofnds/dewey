Lustre is a Gleam-based web framework designed for building HTML templates, single-page applications, and real-time server components. It provides a declarative, functional API for constructing HTML, an Erlang/Elm-inspired state management system, and managed side effects for predictable code. Key features include server-side rendering and universal components allowing components to run in multiple environments. This framework is a developer tool intended for frontend developers building web applications using the Gleam programming language.

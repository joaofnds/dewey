`adr-tools` is a command-line tool written in Shell for managing Architecture Decision Records (ADRs) within software projects. It facilitates the creation, tracking, and maintenance of ADRs as Markdown files, storing them in a designated directory. Key functionalities include initializing an ADR log, creating new ADRs with support for superseding existing records, and providing help documentation. This standalone CLI application is designed for software developers and architects to document and maintain architectural decisions.

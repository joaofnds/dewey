`liquidjs` is a TypeScript library providing a Shopify/GitHub Pages compatible template engine in pure JavaScript, designed for server-side and browser environments. It enables the parsing and rendering of Liquid templates, facilitating the porting of Jekyll, GitHub Pages, and Shopify templates to Node.js applications. Key functionalities include tag and output syntax processing for dynamic content generation. This is a library, offering a JavaScript implementation of the Liquid templating language, intended for frontend developers and server-side rendering within JavaScript projects, and providing a CLI for template processing.

Weaver is a CLI tool for tracing Go program execution using eBPF and uprobes on Linux systems. It enables the inspection of function argument values within running or compiled Go programs.  The tool supports tracing via a function specification file or by package, and currently supports basic data types. Key dependencies include bcc and a Linux kernel version greater than 4.14. Weaver is a developer productivity tool, designed for debugging and understanding Go application behavior by providing insights into function calls and data flow.

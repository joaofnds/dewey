`Postmodern` is a Common Lisp library providing a PostgreSQL programming interface, enabling interaction with PostgreSQL databases. Key features include efficient database communication without foreign libraries, support for UTF-8, a syntax for embedding SQL within Lisp code (S-SQL), and support for prepared statements and stored procedures. It also provides a metaclass for defining database access objects (DAOs). The library leverages Common Lisp and depends on libraries such as `usocket`, `md5`, and `closer-mop`. It is a library and is primarily used by Common Lisp developers needing to interact with PostgreSQL databases.

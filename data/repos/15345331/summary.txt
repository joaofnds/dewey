Bolt is a Go library providing an embedded, ACID-compliant key-value database, designed for projects prioritizing simplicity and speed over the full features of larger database servers. Its core functionality centers on storing and retrieving data with a straightforward API focused on setting and getting values within buckets.  This library supports read-only and read-write transactions, providing features like nested buckets, database backups, statistics collection, and mobile use.  Bolt is a library, serving as an alternative to relational databases, LevelDB, and LMDB, primarily targeting Go developers seeking persistent, embeddable storage within their applications.

`nvim-treesitter-context` is a Neovim plugin providing in-buffer code context display, acting as a lightweight alternative to context.vim. Built using the Janet programming language, it leverages the `nvim-treesitter` plugin for language parsing to display surrounding code blocks above the current cursor position. Key functionalities include customizable display options (line numbers, separator, z-index), multi-window support, and the ability to configure context presentation based on cursor position or the top line of the visible viewport. This plugin is a Neovim plugin primarily for developers using the Neovim text editor.

ZITADEL is an open-source identity and access management (IAM) platform built in Go that provides comprehensive authentication and authorization infrastructure for web applications and services. It implements OpenID Connect, OAuth 2.0, SAML 2.0, and SCIM 2.0 protocols with support for multi-factor authentication including FIDO2/WebAuthn passkeys, OTP, and biometrics. The platform features native multi-tenancy architecture with event sourcing on PostgreSQL, enabling B2B and B2C identity scenarios with self-service user management, role-based access control, and identity brokering. Target users include software developers, DevOps engineers, and system architects building modern applications requiring secure user authentication, SSO capabilities, and scalable identity management with API-first integration patterns.

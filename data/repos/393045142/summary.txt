Rnote is a cross-platform vector-based drawing and note-taking application built with Rust and GTK4, targeting digital education and creative workflows. The application serves students, teachers, and stylus device users who need handwritten note-taking, document annotation, and sketching capabilities. It features pressure-sensitive stylus input with Wacom tablet support, an infinite canvas system, and comprehensive document handling including PDF/SVG import/export functionality. The codebase implements a custom .rnote file format using gzipped JSON, provides multi-platform deployment via Flatpak, Windows installers, and macOS bundles, and includes workspace management for organizing related files. Key technical components include adaptive UI rendering for various screen sizes, real-time stroke processing with configurable pen styles, shape tools with selection/transformation capabilities, and integrated clipboard/drag-and-drop support for seamless workflow integration in educational and creative environments.

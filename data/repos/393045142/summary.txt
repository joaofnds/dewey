Rnote is a standalone, open-source vector-based drawing application for sketching and handwritten note-taking, designed for use with drawing tablets and stylus input. Built with Rust and GTK4, it provides an infinite canvas, support for PDF, bitmap, and SVG import/export, and an adaptive UI. Key features include customizable stroke styles, shape tools, selection tools for content manipulation, document layout options, and stylus button shortcuts. This application targets users needing digital handwriting and annotation capabilities, such as students and educators.

`acts_as_tenant` is a Ruby on Rails library designed to facilitate row-level multi-tenancy in applications sharing a single database. It enables tenant-aware scoping of ActiveRecord models, ensuring data isolation for different tenants, typically identified via subdomains or domains. Core functionalities include setting and managing the current tenant, validating tenant-specific uniqueness, and providing methods for scoping model queries, along with support for background processing libraries like ActiveJob and Sidekiq. This library is a Ruby gem, targeting Rails developers seeking a shared database multi-tenancy solution.

`Level` is a source-available, full-stack SaaS application designed for team communication, optimized for deep work. Built with Elixir/Phoenix on the backend and an Elm single-page application frontend, it leverages GraphQL for data fetching. The application includes features for user authentication, asset storage via AWS S3, and utilizes Postmark for transactional emails. This project aims to provide a self-installable instance and includes Heroku deployment instructions, targeting developers seeking a communication platform.

The `Windows10Debloater` repository is a PowerShell script and GUI application designed to remove pre-installed bloatware, disable telemetry functions, and modify system settings on Windows 10. It provides interactive and silent (SysPrep) modes, offering functionalities such as removing appx packages, disabling scheduled tasks, and modifying privacy-related registry keys.  Built with PowerShell, it leverages both command-line execution and a GUI interface for ease of use. This project is a standalone application intended for system administrators and advanced users seeking to customize and optimize Windows 10 installations.

Terrier is a CLI tool written in Go designed for analyzing container images and running containers by verifying the presence and SHA256 hashes of specific files. It operates within the container and image security domain, enabling users to scan OCI images (from a TAR archive) or running containers to identify the existence of files and validate their integrity against a provided configuration file. Key functionalities include image scanning, container scanning, and the ability to integrate with CI/CD pipelines. This standalone application utilizes Go, and the primary target users are likely DevOps or security engineers focused on supply chain verification and container image security audits.

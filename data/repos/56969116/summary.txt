`ngosang/trackerslist` is a data repository providing frequently updated lists of public BitTorrent trackers. It serves as a resource for BitTorrent users, offering lists categorized by protocol (HTTP, UDP, WebSockets, etc.) and DNS/IP address formats. The project automatically updates the tracker lists and provides mirrored links, including versions suitable for use with I2P networks. This is a data-centric project and a curated collection of tracker URLs.

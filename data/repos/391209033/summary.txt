`soft-serve` is a self-hostable Git server designed for command-line interaction, written in Go. It provides a Git server with features accessible via SSH, HTTP, and the Git protocol, including a terminal user interface (TUI) for browsing repositories, file viewing with syntax highlighting, Git LFS support, and user access control through SSH keys and user tokens. Functionality includes repository management via SSH commands, mirroring capabilities, and support for server-side Git hooks. This standalone application targets developers and system administrators seeking a self-managed Git solution.

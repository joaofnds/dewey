Baton is a command-line interface (CLI) tool written in Go for managing Spotify playback. It provides CLI commands for authorization, device control, playback management (play, pause, next, previous), search functionality with an integrated TUI, volume control, and track information retrieval. The application leverages the Cobra CLI framework and the Gocui TUI library. It allows users to interact with the Spotify Web API, enabling management of playback, including device selection and search.
